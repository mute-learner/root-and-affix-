| PIE 级 | 拉丁线表面形式 | 日耳曼线表面形式 | 代表词 |
|--------|-------------|----------------|--------|
| e 级 | **cap-**（直接）→ capable, capacity | **hab-/hav-** → **have**, heave |  |
| | **-cip-**（复合词弱化）→ anticipate, participate, principal | | |
| | **-cup-**（唇音前缀）→ occupy, recover | | |

> 共 9 种表面形式（cap-, -cip-, -cup-, -ceiv-, capt-, -cept-, -ceit-, hab-/hav-, heav-），来自 3 个 PIE 等级（e / zero / ō）× 2 条语族路径。这是目前笔记里 ablaut 跨度最广的词根——五级开了三级，加上法语二次加工。

## ② 叙事链：cap——从手掌合拢到你的 every word

**这不是一根词根，这是一只手掌合拢的动作。**

PIE **\*kap-** 在六千年前被说出时，指的不是一个抽象概念。它是这样一个画面：伸出手指，触到目标，五指同时向掌心合拢——握住了。一只羊的角被牧人抓住。一把燧石的边缘被工匠捏紧。一根木棍被猎人握在手里。**这是人类最古老、最日常的身体动作之一。** 所以它活了下来——在每一个印欧语族的后代中都活着。[Wikt-PIE][Etym]

---

**拉丁线经历了三次关键扩张。**

**第一次：物理→空间。** 原始意大利人把 PIE \*kap- 继承为 \*kapiō（"我抓住"）。"抓住一个物体"自然延伸到"这个容器能抓住多少东西"——**capax**（能容纳的）。capable 和 capacity 的"容纳→能力"语义迁移，在拉丁语阶段已经完成。罗马工程师计算水渠的容量时用的就是 capax 的派生词。[Etym]

**第二次：空间→心理。** 这是最关键的一步。"手合拢抓住东西"的意象被投射到了看不见的领域——mental grasping。拉丁人开始说 **percipere**（per- 穿透 + capere 抓→彻底抓进意识→感知），说 **concipere**（con- 一起 + capere 抓→把所有经验一起抓到脑子里→构思、酝酿）。[Etym][AH] "理解"在拉丁语里就是"抓进脑子"——这个隐喻深刻到英语里至今还在用：grasp an idea、capture the essence。

**第三次：法律化。** 罗马人对私人财产权有近乎痴迷的讲究。**mancipium**（man- 手 + cap- 抓→用手抓住的东西→正式占有的财产）是罗马法中财产转移的核心概念。从这个法律概念里长出了 **emancipate**——父亲在仪式上对成年儿子"把手拿开"，在法律上释放他。你的"解放"和你的"抓住"，在底层是同一件事的两个面——抓是占有，放手是自由。[Etym]

---

**日耳曼线：同一个 PIE 根，另一个语义方向。**

在拉丁线向"容纳→理解→法律占有"推进的同时，日耳曼人走了另一条路。PIE \*kap- 进入原始日耳曼语时经历了格林定律：**PIE \*k → PGmc \*h**，**PIE \*p → PGmc \*f**（元音间 → \*ƀ → v）。所以拉丁语的 cap- 对应日耳曼语的 **hab-**——首辅音变了，内核没变。[AH][Etym]

PGmc **\*habjaną** 不是"抓一个新东西"，而是"**已经抓住了的东西，继续拿着**"——这是体貌层面的语义选择：拉丁选了抓取的起始瞬间（capture 抓住那一刻），日耳曼选了抓取后的持续状态（**have** 持续持有）。同一个物理动作的两面。

**heave**（抬起重物→需要双手抓住才能做到）、**heavy**（重的→需要双手抓住来搬的）、**haven**（港湾→能"抓住"船只的安全之地）——这些词跟 capture、accept 来自同一个根。你每说一次 "I have it"，你嘴里的 have 讲的还是那个六千年老的动作——抓住，没松手。[Etym]

---

**法语中介：拼写不是装饰，是中世纪发音的化石。**

诺曼征服（1066 年）后，已经经历了拉丁内部元音弱化的 -cip- 和 -cept- 形式通过诺曼法语进入中古英语。在法语中，拉丁短 i（-cip- 中的 i）经历了双元音化——先降为 /e/，再升为 /ei/→ 拼写变成 -ceiv-：recipere → OF receivre → ME receiven → **receive**。拉丁 -pt-（-cept- 中的 -pt-）在法语中 -p- 脱落，只剩 -t-→ 拼成 -ceit-：conceptus → OF conceite → **conceit**。[Etym]

**concept**（直借拉丁，抽象概念）和 **conceit**（经法语，脑子里"抓"了太多自己→自负）——同一个拉丁词 concipere，走两条不同的历史路径进入英语，变成了两个意思完全不同的现代词。这是历史语言学能给的最好的警句：同一个源，路径决定意义。

**captiare → catch / chase / purchase** 是另一条法语路径——拉丁频繁体 captiare（反复抓）在法语中经历了腭音化（-pti- → -ch-）：captiare → OF cachier → catch（日常用语），OF chacier → chase（追赶），pro- + captiare → OF purchacier → purchase（购买——字面"追着买"）。拉丁的 -pt- 在这里被磨成了一口法语腭擦音，把 lab 的形迹彻底抹掉了。[Etym]

---

**文艺复兴：批量铸造的学术大军。**

15-17 世纪，英国学者翻开拉丁词典，发现 capere 可以跟几乎任何前缀组合成新的抽象名词。他们一次性借入了：inception（in- + cept-→开始）、precept（pre- + cept-→提前拿走的教诲→戒律）、intercept（inter- + cept-→从中间截住）、susceptible（sub- + cept-→容易被拿下的）。这些词在口语里几乎不用，属于拉丁学术层。[Etym]

但这恰恰是 cap 词根最强大的地方：**它同时覆盖了英语里频率最高的词（have）和最学术的词（emancipate），两者的底层是同一个身体动作。**

## ③ PIE 元音变换（ablaut）——为什么 cap 的变体长这样

叙事链里的那些表面形式，之所以不一样，根本原因早在拉丁语和日耳曼语分化之前就种下了——PIE 的重音系统决定了元音等级。

**ablaut 是什么。** PIE 的基本元音是短 \*e。同一个词根，当重音位置移动时，\*e 发生系统性变化：重音在本音节 → e 级；重音后移 → 元音消失（零级）；重音前移 → e 变 o（o 级）；词末响音前 → 元音拉长（长级）。输入是重音位置，输出是元音等级。[Wikt-PIE]

**\*keh₂p- 的五级完整展开：**

| 级 | PIE 形式 | 重音条件 | 拉丁后代 | 日耳曼后代（k→h, p→f） |
|----|---------|---------|---------|---------------------|
| e 级 | \*keh₂p- → \*kap- | 重音在本音节 | **capere** → capable, accept, receive | **\*habjaną** → have |
| o 级 | \*koh₂p- | 重音前移 | **cōp-** → couple, copula | **\*hōb-** → behoof |
| 零级 | \*kh₂p- | 重音后移→e 消失 | **captus** → capture, captive, catch | **\*haftaz** → haft, heft |
| 长 ē | \*kēh₂p- | 词末响音前 | **cēpī**（拉丁完成时，未借入英语） | — |
| 长 ō | \*kōh₂p- | 词末响音前 | — | **\*hōf-** → behoove, heave, heavy |

**喉音 \*h₂ 的关键角色。** 为什么 e 级拼成 \*keh₂p- 但实际是 \*kap-？因为喉音 \*h₂ 把它前面的 \*e 染成了 \*a。所以 e 级不是 \*kep- 而是 \*kap-——喉音改了元音颜色，但 ablaut 等级不变。同样，o 级 \*koh₂p- 中 \*o + h₂ 拉长后变成拉丁 cōp- 和日耳曼 hōf-。[Wikt-PIE]

**零级的具体走法。** \*e 消失后，词根变成 \*kh₂p-（只有辅音骨架），喉音 \*h₂ 承担音节，像一个短促的"uh"。到了拉丁语：\*kh₂p- + 分词后缀 -tus = captus（"被抓住的东西"）。到了日耳曼语：\*kh₂p- + -tós = \*haftaz（格林定律 k→h, p→f）。同一条零级，两条语族走了不同的表面拼写。[Etym][AH]

**不是所有级都在英语里有后代。** e 级和零级是最大宗的——90% 的 cap 词来自这两个级。o 级只有 couple/copula（经拉丁）和 behoof（经日耳曼，已废弃）。长 ē 级根本没进英语（拉丁完成时 cēpī 被日耳曼/法语自己的时态替代了）。长 ō 级进了日耳曼线——heave、heavy 里面的 -ea- 追溯到 \*ō。

**两层因果框架（这是核心认知）：**
1. **深度原因**（PIE 时代）：capture 来自零级 \*kh₂p-，have 来自 e 级 \*kap-。六千年重音位置就已经决定了元音不同。
2. **表层原因**（语族分化后）：零级加了拉丁分词后缀 -tus → capt-；e 级经过格林定律 → hab-。两层叠加，产生了 cap- 和 hav- 这种看起来完全不像的表面形式。

|------|------|-------|------|

| 希腊语族 | 古希腊语 | **kṓpē**（κώπη） | 手柄、船桨（手握的部分） |
| 凯尔特语族 | 古爱尔兰语 | **cúan** | 港湾 |

| 阿尔巴尼亚语 | 阿尔巴尼亚语 | **kap** | 抓、逮 |
| 印度-伊朗语族 | 梵语 | **kapā́la-** | 碗、头骨 |

## ⑥ 附注

**etymological debate — \*kap- 的学术争议。**

不是所有学者都认同 \*kap- 的纯 PIE 身份。Pokorny (1959) 重建为纯 \*kap-（无喉音）。Rix / LIV² (2001) 重建为 \*keh₂p-——喉音解释了希腊语 kṓpē 中的长 ō。de Vaan (2008) 更保守：如果去掉希腊语的连接（Greek káptō "狼吞虎咽" 和 Latin capere 可能语义不匹配），剩下的意大利和日耳曼形式可能是从欧洲基质语（substrate）借来的。对学习者来说，这些争议不影响词根的实际使用——不管你叫它 \*kap- 还是 \*keh₂p-，现代英语里的运作逻辑不变。学者吵架属于 Footnotes。[Wikt-PIE]

**\*kap-（抓）和 \*kaput-（头）的关系。** 两个根长得太像，有些学者怀疑它们本来就是同一个——"头"就是"用来抓/咬/叼东西的身体部位"。梵语 kapā́la- 既指"头骨"也指"碗"（抓东西的容器），是这一假说的核心证据。但假说没有被广泛接受，目前的安全做法是分成两个根处理。[AH]
