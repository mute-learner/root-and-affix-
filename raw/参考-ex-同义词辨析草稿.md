# ex- 词根同义词辨析草稿

格式来自 [[参考-同义词辨析示例]]——一段话，不表格。

---

exit 和 leave 都作"离开"解。但是词根不一样：exit 的 ire 意为 to go（走），构词 ex- + ire = 走出去；leave 的词根 *leip- 意为 to leave behind（留下、离开）。所以，exit 偏重于从一个有边界的空间里走出去——门、建筑、系统；而 leave 侧重于离开任何地方，不强调边界感。

exclude 和 omit 都作"不包含"解。但是词根不一样：exclude 的 claudere 意为 to close（关闭），构词 ex- + claudere = 关在外面；omit 的 mittere 意为 to send（送），构词 ob- + mittere = 放走、放过。所以，exclude 偏重于主动关在门外——有意图的排斥；而 omit 侧重于被动漏掉——非故意或为省空间。

export 和 import 都作"运输货物跨国"解。但是词根一样（portare = to carry），前缀相反：ex- 意为向外，im- 意为向内。所以，export 偏重于运出去；import 侧重于运进来。记住方向：ex 是 exit 的方向。

extract 和 remove 都作"取出"解。但是词根不一样：extract 的 trahere 意为 to draw, to pull（拉、拖），构词 ex- + trahere = 拉出来；remove 的 movere 意为 to move（移动），构词 re- + movere = 移回去、移开。所以，extract 偏重于从内部用力拉出来——有阻力、有过程，如拔牙；而 remove 侧重于随手拿开——轻松、无阻力，如拿走杯子。

extend 和 expand 都作"扩大"解。但是词根不一样：extend 的 tendere 意为 to stretch（伸展），构词 ex- + tendere = 伸出去；expand 的 pandere 意为 to spread（铺开），构词 ex- + pandere = 铺开出去。所以，extend 偏重于沿一个方向拉长——线性延伸，如道路延伸、期限延长；而 expand 侧重于向四面八方展开——面积或体积膨胀，如气球膨胀、业务扩张。

expand 和 swell 都作"变大"解。但是词根不一样：expand 的 pandere 意为 to spread（铺开）；swell 的词根 *swel- 意为 to swell（膨胀）。所以，expand 偏重于有序地向各方向展开——可控的扩张；而 swell 侧重于因内部压力无序地鼓起来——暂时的、不可控的肿胀。

exceed 和 surpass 都作"超过"解。但是词根不一样：exceed 的 cedere 意为 to go, to yield（走、去），构词 ex- + cedere = 走出去、走到界限之外；surpass 的 passer 意为 to pass（通过），构词 sur- + passer = 从上面通过。所以，exceed 偏重于超过客观数值或限度——中性的度量，如超速、超预算；而 surpass 侧重于在品质上超越他人——正面的评价，如超越对手、超越自我。

express 和 convey 都作"传达"解。但是词根不一样：express 的 pressare 意为 to press（压），构词 ex- + pressare = 压出来；convey 的 via 意为 way（道路），构词 con- + via = 一起走一条路→搬运。所以，express 偏重于从内部压出来——直接的、来自内心的表达；而 convey 侧重于通过媒介搬运传递——间接的、经过通道的传达。

emit 和 release 都作"放出"解。但是词根不一样：emit 的 mittere 意为 to send（送），构词 e- + mittere = 送出去；release 的 laxare 意为 to loosen（放松），构词 re- + laxare = 松开、放开。所以，emit 偏重于从内部持续向外发送——光、热、气体、声音；而 release 侧重于一次性释放被禁锢的东西——如释放囚犯、松开气球。

erupt 和 explode 都作"爆发"解。但是词根不一样：erupt 的 rumpere 意为 to break（打破），构词 e- + rumpere = 破出来；explode 的 plaudere 意为 to clap（拍打），构词 ex- + plaudere = 拍打驱逐→炸开。所以，erupt 偏重于从内部冲破表层——火山式的、有持续性的喷发；而 explode 侧重于一次性剧烈炸开——炸弹式的、瞬间的爆裂。

evolve 和 develop 都作"发展"解。但是词根不一样：evolve 的 volvere 意为 to roll（卷、滚动），构词 e- + volvere = 向外卷开→展开；develop 的 voloper 意为 to wrap（包裹），构词 des- + voloper = 解开包裹。所以，evolve 偏重于从内部按内在逻辑自然展开——像卷轴一层层展开，不可加速，如物种进化；而 develop 侧重于经外部力量推动成长——可被策划和加速，如开发项目。

effect 和 affect 都作"影响"解。但是词根一样（facere = to do, to make），前缀不同：ef-（=ex-）意为向外，af-（=ad-）意为朝向。effect = 做出来→产生的结果；affect = 对着…做→施加影响。所以，effect 偏重于影响的结果——名词为主，表示"效果"；而 affect 侧重于施加影响的过程——动词为主，A 影响 B。effect 做动词时约等于 accomplish——效果是你主动"做出来"的，而不是被动发生的。

exclaim 和 shout 都作"喊叫"解。但是词根不一样：exclaim 的 clamare 意为 to cry out（喊叫），构词 ex- + clamare = 彻底喊出来→把声音全部放出来；shout 的词根 *skeut- 意为 to shout（喊叫）。所以，exclaim 偏重于情绪驱动、带有具体内容地喊出话来——后面跟引语，如 "What a view!" he exclaimed；而 shout 侧重于纯粹提高音量——可以不说话，只求声音大。

excite 和 stimulate 都作"激起"解。但是词根不一样：excite 的 citare 意为 to summon, to rouse（召唤、激起），构词 ex- + citare = 彻底激起；stimulate 的词根 stimulus 意为 goad（刺棒——赶牛用的尖棍）。所以，excite 偏重于剧烈地唤起情绪反应——心跳加速、热血沸腾；而 stimulate 侧重于温和地激活功能——像刺棒轻戳，如刺激经济、启发思维。

extreme 和 severe 都作"极其"解。但是词根不一样：extreme 的 extr- 意为 outer（更外的），构词 extr- + -ēmus（最高级后缀）= 最外面的→到了尽头的；severe 的词根 severus 意为 serious, strict（严肃、严厉）。所以，extreme 偏重于在连续体的最边缘——客观位置，度量上走到了极限；而 severe 侧重于感受上非常糟糕或强烈——主观判断，如严重的伤势。

strange 和 foreign 都作"外来"解。但是词根不一样：strange 的 extra 意为 outside（在外面），构词 extra → 古法语 estrange → strange = 来自外地的；foreign 的 foris 意为 outside the door（门外）。所以，strange 偏重于因不熟悉而产生的心理感受——"不对劲"、"奇怪"；而 foreign 侧重于来自另一个国家的客观事实——不附带心理评价。一个 foreign student 不一定 strange，一个 strange idea 不是 foreign。
