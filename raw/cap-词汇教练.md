## 模块 1：基础信息

- **变体总览**：这个词根在英语里有 <span style="color:red">8 种表面形式</span>，来自 <span style="color:red">3 个 PIE 元音等级 × 2 条语族路径</span>。机制见[[cap-词源史官]]。

  | 级 | 拉丁线 | 日耳曼线 | 代表词 |
  |----|-------|---------|--------|
  | e 级 | **cap-** → 直接派生 | hab-/hav- → **have**, heave | |
  | | **-cip-** → 复合词弱化 | | |
  | | **-cup-** → 唇音前缀 | | |
  | | **-ceiv-** → 法语双元音化 | | |
  | 零级 | **capt-** → 过去分词直接 | haft- → **haft**, heft | |
  | | **-cept-** → 过去分词复合词 | | |
  | | **-ceit-** → 法语简化 | | |
  | | **catch/chase** → 法语腭音化 | | |
  | ō 级 | cōp- → couple | heav- → **heave, heavy** | |

- **核心本义**：<span style="color:blue">用手抓住、拿住、握在手中</span>

- **同源词概览**：
  - 意大利语族（拉丁 → 借词）：Etymonline 列出的 60+ 词，见模块 4 全文
  - 日耳曼语族（共享 PIE，平行演化，非借词）：have, heave, heavy, haven, haft, hawk, behave, behoof, behoove, heft

- **语域层**：绝大多数属于拉丁学术层或法语中性层。只有日耳曼同源词 have/heave 属于日耳曼日常层。

- **词频层**：
  - 核心高频（第 1-2 层）：have, accept, receive, except, catch
  - 中高频（第 3-4 层）：concept, capture, capable, capacity, perceive
  - 中低频（第 5-6 层）：conceive, anticipate, participate, intercept
  - 学术低频（第 7+ 层）：emancipate, capacious, incipient, precept

## 模块 2：词源演化链

<span style="color:red">两条平行线</span>：拉丁 cap- 线走"拿→抓→理解"的语义方向；日耳曼 hab- 线走"拿→持有一拥有"的语义方向。同一个 PIE 祖形，六千年后在英语里重逢。

## 模块 3：语义锚点 + 知识挂点

**知识挂点**：
- 你认识 **have**——它就是 cap- 在日耳曼语族的后代。格林定律 PIE \*k → \*h，所以 cap- 对应 hab-。have 和 capture 是同一个 PIE 祖形。
- 学过法语的话，**recevoir**（<span style="color:blue">收到</span>）和英语 receive 是同一批诺曼征服搬运的词。

<span style="color:red">底层核心逻辑</span>：这组词的所有现代义，回到同一个身体经验——<span style="color:blue">"伸手、合拢、把东西收进自己的控制范围内"</span>。物理动作（capture 抓俘虏）→ 空间容纳（capacity 能装多少）→ 心智理解（conceive 在心里形成想法）→ 社会接纳（accept 接受邀请）→ 所有权占有（have 拥有）。五个维度，同一个根。

<span style="color:red">易误解盲区</span>：
- cap（<span style="color:blue">帽子</span>）不来自 capere——来自拉丁 caput（<span style="color:blue">头</span>）。长得一样，完全不同的根。
- captain（<span style="color:blue">船长</span>）、capital（<span style="color:blue">资本</span>）不来自 capere——来自 PIE \*kaput-（<span style="color:blue">头</span>）。

## 模块 4：高频词汇拆解

### 词族树

```
                    PIE *kap-（抓、握住）
                    /                \
          拉丁线（capere）          日耳曼线（*habjaną）
          /    |    |    \                 |
    物理抓  空间容纳 心理抓  社会接纳    持有一拥有
       |      |      |       |            |
    capture  capacity concept  accept    have
    captive  capable  perceive receive  heave
    catch    capacious conceive except  heavy
```

---

### A 组：cap- 直接派生词

**1. capable** adj. <span style="color:blue">有能力的；能胜任的</span>
- 构词：cap（<span style="color:blue">容纳</span>）+ -able（<span style="color:blue">能够被…的</span>）
- 字面义：<span style="color:red">能够容纳（更多事物）的</span>
- 实际义：有能力做某事
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| be capable of + 动名词 | <span style="color:blue">能够做某事</span> |
| a capable + 职业 | <span style="color:blue">一位能干的…</span> |

> vs **able**：able 更通用（be able to do），capable 强调内在能力、有潜力。capable of murder（<span style="color:blue">有杀人能力</span>）不用 able。

---

**2. capacity** n. <span style="color:blue">容量；能力；身份</span>
- 构词：capac（=capax <span style="color:blue">能容纳的</span>）+ -ity（<span style="color:blue">抽象名词后缀</span>）
- 字面义：<span style="color:red">容纳的能力（抽象化为数量概念）</span>
- 实际义：1）物理容积；2）生产能力；3）心智能力；4）法律身份
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| the capacity of + 容器 | <span style="color:blue">…的容量</span> |
| the capacity to do sth. | <span style="color:blue">做某事的能力</span> |
| at full capacity | <span style="color:blue">满载、全力运行</span> |
| in one's capacity as + 职务 | <span style="color:blue">以…的身份</span> |

---

**3. capacious** adj. <span style="color:blue">宽敞的；容积大的</span>
- 构词：capac（<span style="color:blue">能容纳的</span>）+ -ious
- 字面义：<span style="color:red">能装很多东西的</span>
- 词频层：第 8 层（低频学术词）

---

### B 组：-cept- 过去分词派生词

**4. accept** v. <span style="color:blue">接受；同意；接纳</span>
- 构词：ac-（=ad- <span style="color:blue">朝向</span>）+ cept（<span style="color:blue">被拿</span>）
- 字面义：<span style="color:red">把对方递来的东西拿过来</span>
- 实际义：1）接受（礼物/邀请/建议）；2）接纳（成员）；3）承认（现实/责任）
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| accept + 礼物/邀请/建议 | <span style="color:blue">接受</span> |
| accept + responsibility/blame | <span style="color:blue">承担责任</span> |
| accept that + 从句 | <span style="color:blue">承认（某种现实）</span> |

> vs **receive**：<span style="color:red">accept = 主动接受，receive = 被动收到。</span> 你 receive 了一封邮件（它进了收件箱），但你可能没有 accept 它的内容。

---

**5. except** prep./conj. <span style="color:blue">除了…之外</span> v. <span style="color:blue">不包括</span>
- 构词：ex-（<span style="color:blue">向外</span>）+ cept（<span style="color:blue">被拿</span>）
- 字面义：<span style="color:red">从集合中拿一个出来放在外面→不包含在内</span>
- 词频层：第 1 层

| 搭配模式 | 说明 |
|---------|------|
| except + 名词/代词 | <span style="color:blue">除了…</span> |
| except for + 名词 | <span style="color:blue">除了（有微调语气）</span> |

---

**6. concept** n. <span style="color:blue">概念；想法</span>
- 构词：con-（<span style="color:blue">共同、一起</span>）+ cept（<span style="color:blue">被拿</span>）
- 字面义：<span style="color:red">把多个经验"一起抓进"脑子里形成的抽象物</span>
- 词频层：第 3 层

| 搭配模式 | 说明 |
|---------|------|
| the concept of + 抽象名词 | <span style="color:blue">…的概念</span> |
| grasp/understand a concept | <span style="color:blue">理解一个概念</span> |

---

**7. intercept** v. <span style="color:blue">拦截；截获</span>
- 构词：inter-（<span style="color:blue">在…之间</span>）+ cept（<span style="color:blue">被拿</span>）
- 字面义：<span style="color:red">在物体到达目的地之前，从中间把它截住</span>
- 词频层：第 5 层

| 搭配模式 | 说明 |
|---------|------|
| intercept + 武器/信号 | <span style="color:blue">拦截</span> |
| intercept + 通信/情报 | <span style="color:blue">截取</span> |

---

**8. inception** n. <span style="color:blue">开端；起初</span>
- 构词：in-（<span style="color:blue">进入</span>）+ cept（<span style="color:blue">拿</span>）+ -ion
- 字面义：<span style="color:red">开始拿→着手</span>
- 词频层：第 5 层

---

**9. precept** n. <span style="color:blue">戒律；准则</span>
- 构词：pre-（<span style="color:blue">提前</span>）+ cept（<span style="color:blue">拿</span>）
- 字面义：<span style="color:red">提前"拿走"的教诲</span>
- 词频层：第 7 层

> vs **principle**：precept = 别人教你的规则（<span style="color:blue">外部</span>），principle = 你自己信的原则（<span style="color:blue">内部</span>）。

---

**10. susceptible** adj. <span style="color:blue">易受影响的；敏感的</span>
- 构词：sus-（=sub- <span style="color:blue">向下</span>）+ cept（<span style="color:blue">拿</span>）+ -ible
- 字面义：<span style="color:red">容易被"拿下"的</span>
- 词频层：第 5 层

| 搭配模式 | 说明 |
|---------|------|
| be susceptible to + 疾病/影响 | <span style="color:blue">易受…影响</span> |

---

### C 组：capt- 过去分词直接派生词

**11. capture** v./n. <span style="color:blue">捕获；俘获；占领；捕捉</span>
- 构词：capt（<span style="color:blue">被抓住的</span>）+ -ure（<span style="color:blue">名词后缀</span>）
- 字面义：<span style="color:red">被抓住的行为/结果</span>
- 词频层：第 4 层

| 搭配模式 | 说明 |
|---------|------|
| capture + 人或物 | <span style="color:blue">抓获、俘获</span> |
| capture + 画面/镜头/气氛 | <span style="color:blue">捕捉（引申义）</span> |
| capture + the attention of | <span style="color:blue">吸引…的注意力</span> |

> vs **catch**：<span style="color:red">capture = 正式捕获（军事/数据/摄影），catch = 日常抓住。</span> 你 catch a cold（<span style="color:blue">感冒</span>），不会 capture a cold。

---

**12. captive** n. <span style="color:blue">俘虏</span> adj. <span style="color:blue">被俘的；被迷住的</span>
- 构词：capt（<span style="color:blue">被抓住</span>）+ -ive
- 字面义：<span style="color:red">被抓住的（人）</span>
- 词频层：第 5 层

| 搭配模式 | 说明 |
|---------|------|
| take/hold sb. captive | <span style="color:blue">俘虏某人</span> |
| a captive audience | <span style="color:blue">被迫的听众（想走也走不了）</span> |

---

**13. captivate** v. <span style="color:blue">迷住；使入迷</span>
- 构词：captiv（=captive <span style="color:blue">被抓住的</span>）+ -ate
- 字面义：<span style="color:red">使某人"被抓住"</span>
- 实际义：深深吸引某人（注意力被完全"抓走"了）
- 词频层：第 6 层

> vs **fascinate**：captivate = 抓住了注意力（<span style="color:blue">主动"抓"的意象</span>），fascinate = 让人着迷（<span style="color:blue">被"魔力"镇住的意象</span>）。

---

**14. caption** n. <span style="color:blue">标题；字幕；说明文字</span>
- 构词：capt（<span style="color:blue">抓住</span>）+ -ion
- 字面义：<span style="color:red">抓住（内容要点）的东西→标题</span>
- 词频层：第 4 层

---

**15. captor** n. <span style="color:blue">捕获者；劫持者</span>
- 构词：capt（<span style="color:blue">抓</span>）+ -or（<span style="color:blue">施动者</span>）
- 词频层：第 6 层

---

**16. captious** adj. <span style="color:blue">吹毛求疵的</span>
- 字面义：<span style="color:red">喜欢"抓住"别人的小毛病→挑剔的</span>
- 词频层：第 8 层（低频）

---

### D 组：-ceiv- 法语路径派生词

**17. receive** v. <span style="color:blue">收到；接待；接收</span>
- 构词：re-（<span style="color:blue">往回</span>）+ ceive（=-cip-，<span style="color:blue">拿，经法语</span>）
- 字面义：<span style="color:red">拿回来</span>
- 词频层：第 1 层

| 搭配模式 | 说明 |
|---------|------|
| receive + 物品/礼物/信件 | <span style="color:blue">收到</span> |
| receive + 人 + warmly/coldly | <span style="color:blue">接待态度</span> |
| receive + treatment/training | <span style="color:blue">接受（治疗/训练）</span> |

---

**18. receipt** n. <span style="color:blue">收据；收到</span>
- 构词：re- + ceit（=-cept-，<span style="color:blue">经法语的过去分词</span>）
- 字面义：<span style="color:red">收到的凭证</span>
- 词频层：第 4 层

---

**19. conceive** v. <span style="color:blue">构思；设想；怀孕</span>
- 构词：con-（<span style="color:blue">一起</span>）+ ceive（=-cip-，<span style="color:blue">拿，经法语</span>）
- 字面义：<span style="color:red">把零散的东西一起抓到脑子里→在心里合成一整个</span>
- 词频层：第 4 层

| 搭配模式 | 说明 |
|---------|------|
| conceive of + 名词 | <span style="color:blue">想象、设想</span> |
| conceive + a plan/idea | <span style="color:blue">构思一个计划/想法</span> |
| conceive + a child | <span style="color:blue">怀孕</span> |

> vs **imagine**：<span style="color:red">conceive = 在脑子里孕育成形（从无到有），imagine = 在脑子里浮现画面。</span> 你 conceive a plan（<span style="color:blue">构思一套方案</span>），imagine a scene（<span style="color:blue">想象一个场景</span>）。

---

**20. concept** — 见 B 组 #6（直借拉丁）

**21. conceit** n. <span style="color:blue">自负；别出心裁的比喻</span>
- 构词：con- + ceit（<span style="color:blue">经法语过去分词</span>）
- 字面义：<span style="color:red">脑子里"抓"了太多自己→自负</span>
- 词频层：第 6 层
- <span style="color:red">concept（直借拉丁，抽象概念）和 conceit（经法语，自负）——同一个拉丁词两个路径两个意思。</span>

---

**22. perceive** v. <span style="color:blue">感知；察觉；认为</span>
- 构词：per-（<span style="color:blue">彻底、穿透</span>）+ ceive（=-cip-，<span style="color:blue">拿</span>）
- 字面义：<span style="color:red">完全拿进来→把外界的信号完整地抓进意识</span>
- 词频层：第 4 层

| 搭配模式 | 说明 |
|---------|------|
| perceive + 事物 + as + 特征 | <span style="color:blue">把…视为…</span> |
| be perceived to be + 特征 | <span style="color:blue">被认为是…</span> |

> vs **notice**：<span style="color:red">perceive = 深度感知并理解，notice = 注意到表面现象。</span> 你 notice 有人在看你（<span style="color:blue">感官捕捉</span>），你 perceive 他怀有敌意（<span style="color:blue">深层判断</span>）。

---

**23. deceive** v. <span style="color:blue">欺骗</span>
- 构词：de-（<span style="color:blue">离开正道</span>）+ ceive（<span style="color:blue">拿</span>）
- 字面义：<span style="color:red">把人从正路上"拿开"→引入歧途</span>
- 词频层：第 4 层

| 搭配模式 | 说明 |
|---------|------|
| deceive + 人 + into + 动名词 | <span style="color:blue">骗某人做某事</span> |

---

**24. deceit** n. <span style="color:blue">欺骗；欺诈</span>
- 构词：de- + ceit（<span style="color:blue">经法语过去分词</span>）
- 词频层：第 5 层

---

### E 组：-cip- 复合词派生

**25. anticipate** v. <span style="color:blue">预期；预先做</span>
- 构词：anti-（=ante- <span style="color:blue">在…之前</span>）+ cip（<span style="color:blue">拿</span>）+ -ate
- 字面义：<span style="color:red">提前拿→在事情发生之前就把它"抓"进计划里</span>
- 词频层：第 4 层

| 搭配模式 | 说明 |
|---------|------|
| anticipate + 名词 | <span style="color:blue">预期某事</span> |
| anticipate + a problem/need | <span style="color:blue">预先应对问题/需求</span> |

> vs **expect**：<span style="color:red">anticipate = 预见并提前准备（带行动），expect = 等待事情发生（可以不带行动）。</span> 你 expect 下雨（<span style="color:blue">觉得会下</span>），你 anticipate 下雨带了伞（<span style="color:blue">准备了</span>）。

---

**26. participate** v. <span style="color:blue">参与；参加</span>
- 构词：parti-（<span style="color:blue">部分</span>）+ cip（<span style="color:blue">拿</span>）+ -ate
- 字面义：<span style="color:red">拿走一部分→分到一部分责任或行动</span>
- 词频层：第 4 层

| 搭配模式 | 说明 |
|---------|------|
| participate in + 活动/项目 | <span style="color:blue">参与…</span> |

---

**27. emancipate** v. <span style="color:blue">解放</span>
- 构词：e-（=ex- <span style="color:blue">向外</span>）+ manu-（<span style="color:blue">手</span>）+ cip（<span style="color:blue">拿</span>）+ -ate
- 字面义：<span style="color:red">把手从束缚中"向外拿"出来</span>
- 罗马法背景：父亲"把手拿开"释放成年儿子→法律上的解放
- 词频层：第 7 层

## 模块 5：终极避坑 + 词间连接

### 拼写变化规则

拉丁内部元音弱化：cap- 在复合词中 → -cip-（非唇音前缀）或 -cup-（唇音前缀）。法语化拼写：-cip- → -ceiv-（双元音化），-cept- → -ceit-（-p-脱落）。

### <span style="color:red">假词根甄别</span>

以下词长得像但来自 PIE \*kaput-（<span style="color:blue">头</span>），不是 \*kap-（<span style="color:blue">抓</span>）：

| 单词 | 真实来源 | 本义 |
|------|---------|------|
| **captain**（<span style="color:blue">船长</span>）| PIE \*kaput- | <span style="color:blue">"头领"→首领</span> |
| **capital**（<span style="color:blue">资本/首都</span>）| PIE \*kaput- | <span style="color:blue">"头部的"→首位的</span> |
| **chapter**（<span style="color:blue">章节</span>）| PIE \*kaput- | <span style="color:blue">"小头"→章节标题</span> |
| **cap**（<span style="color:blue">帽子</span>）| 晚期拉丁 cappa | <span style="color:blue">盖头的东西</span> |
| **decapitate**（<span style="color:blue">斩首</span>）| de- + caput- | <span style="color:blue">把"头"去掉</span> |

<span style="color:red">快速鉴别法</span>：词义跟"头、首要、章节、帽子"有关 → \*kaput-（<span style="color:blue">头</span>）。跟"抓、拿、接收、理解"有关 → \*kap-（<span style="color:blue">抓</span>）。

### <span style="color:red">词间近义辨析汇总</span>

| 高频词 | vs | 低频/同义 | 核心区别 |
|--------|----|----------|---------|
| accept | vs | receive | accept = <span style="color:blue">主动接受</span>，receive = <span style="color:blue">被动收到</span> |
| capture | vs | catch | capture = <span style="color:blue">正式捕获</span>，catch = <span style="color:blue">日常抓住</span> |
| perceive | vs | notice | perceive = <span style="color:blue">深层感知理解</span>，notice = <span style="color:blue">表面注意</span> |
| conceive | vs | imagine | conceive = <span style="color:blue">脑子里孕育成形</span>，imagine = <span style="color:blue">浮现画面</span> |
| anticipate | vs | expect | anticipate = <span style="color:blue">预见并准备</span>，expect = <span style="color:blue">等待发生</span> |
| emancipate | vs | free | emancipate = <span style="color:blue">法律/制度解放</span>，free = <span style="color:blue">日常释放</span> |
| chase | vs | pursue | chase = <span style="color:blue">物理追赶</span>，pursue = <span style="color:blue">抽象追求</span> |
| principal | vs | principle | -pal = <span style="color:blue">主要的/校长</span>，-ple = <span style="color:blue">原则</span> |
| concept | vs | conceit | concept = <span style="color:blue">抽象概念（直借拉丁）</span>，conceit = <span style="color:blue">自负（经法语）</span> |
| recover | vs | recuperate | recover = <span style="color:blue">泛指恢复</span>，recuperate = <span style="color:blue">专指身体休养</span> |

### 语域使用避坑

- **capture**：技术文档和军事语境 OK，日常"take a photo"比"capture an image"自然。
- **emancipate**：高度正式/学术词——日常说 free 或 liberate。
- **perceive**：口语中"I perceive..."偏书面——日常用"I notice..."或"I can tell..."。
- **conceive**：表"怀孕"是相当临床/正式的说法——日常说"get pregnant"或"expecting"。
- **capacious**：日常用 spacious，capacious 几乎只在文学/学术文本里见到。

## 模块 6：记忆口诀

<span style="color:red">cap 是抓，抓来抓去全用它：</span>

往前抓 accept（接受），往外抓 except（除外），
中间截 intercept（拦截），一起抓 concept（概念），
往回拿 receive（收到），彻底抓 perceive（感知），
提前抓 anticipate（预期），分着抓 participate（参与），
唇音变 occupy（占据），法语磨成 catch（抓住）。
日耳曼线 have（拥有）——抓住了不松手，就是你的。
