## 模块 1：基础信息

### 变体总览

in- 在英语里有 <span style="color:red">4 种表面形式</span>，由后接辅音的同化决定。但这 4 种形式背后是 <span style="color:red">2 个完全不同的 PIE 词根</span>——它们碰巧长得一模一样。

| 形式 | 出现条件 | 用于 in-向内 | 用于 in-否定 |
|------|---------|-------------|-------------|
| **in-** | 元音或多数辅音前 | invade, include, inspect | inactive, invisible, insane |
| **im-** | b, m, p 前 | import, impose, immigrate | impossible, immoral, immature |
| **il-** | l 前 | illuminate, illustrate | illegal, illiterate, illogical |
| **ir-** | r 前 | irrigate, irradiate | irregular, irresponsible, irrelevant |

<span style="color:red">同样四个形式，两个不相关的源头。</span>这是整个英语前缀系统里最容易造成混淆的事实。

### 核心本义

- **in-（向内）**：<span style="color:blue">从外部进入内部——跨过边界，到里面去</span>。PIE **\*en**（<span style="color:blue">在…里面</span>）。[Etym] [Wikt-PIE]
- **in-（否定）**：<span style="color:blue">不、无、相反</span>。PIE **\*ne**（<span style="color:blue">不</span>）→ 零级 **\*n̥-** → 拉丁 in-。[Etym] [Wikt-PIE]

### 同源词概览

| 语族 | in-向内（PIE \*en） | in-否定（PIE \*ne） |
|------|-------------------|-------------------|
| 拉丁线（借词） | in-/im-/il-/ir-（绝大多数英语 in-词） | in-/im-/il-/ir-（绝大多数否定形容词） |
| 法国线（借词） | en-/em-（enquire, ensure, employ）——同一拉丁 in- 经法语变形 | 无（法国线否定词走其他前缀） |
| 希腊线（借词） | en-/em-（enthusiasm, embryo）——同一 PIE \*en 的另一分支 | a-/an-（amoral, anarchy）——同一 PIE \*ne 的另一分支 |
| 日耳曼线（本土） | **in** 独立介词/副词——英语自己的血肉 | **un-**（unhappy, undo）——英语自己的血肉，同源！ |
| 日耳曼线（本土，已灭绝） | 古英语 on-（onliehtan "enlighten"）——<span style="color:red">全部已死</span> | 无 |

<span style="color:red">最重要的同源事实：</span>英语 in-（否定）和英语 un- 来自 <span style="color:red">同一个 PIE 词根 *ne</span>。in- 走了拉丁路线，un- 走了日耳曼路线。六千年后它们在英语里会师，但没人觉得它们是亲戚。

### 语域层

- **in-（向内）**：绝大多数属于拉丁学术层。import、include、invade 属于日常层。inquire/ensure 经法语（en-）属于法语中性层。
- **in-（否定）**：几乎全属于拉丁学术层。口语中表示否定倾向于用 un- 或 not——「It's not possible」而非「It's impossible」（虽然两个都对，但 not 更日常）。

### 词频层

**in-（向内）：**
- 核心高频（第 1-2 层）：import, include, increase, involve, invest, intend, insist, inspire, indicate
- 中高频（第 3-4 层）：invade, inject, insert, inspect, impose, intense, initiate, inquire, infect, inflate, illustrate
- 中低频（第 5-6 层）：illuminate, impulse, inhabit, innate, immigrate, imprison, inaugurate, inherent, influx, irrigate, innovate, implant

**in-（否定）：**
- 核心高频（第 1-2 层）：impossible, independent, invisible, illegal, irregular, incomplete
- 中高频（第 3-4 层）：infinite, insufficient, inaccurate, insane, inadequate, incapable, inevitable, immoral, immortal, irresponsible, irrelevant
- 中低频（第 5+ 层）：indefinite, impartial, illiterate, illogical, irresistible, infamous, illegitimate, immature

## 模块 2：词源演化链

<span style="color:red">in- 不是动词词根——它是介词/小品词，和 ex- 同类型。</span>两个 in- 各自有一条独立的演化链。

### in-（向内）演化链

| 时期 | 形式 | 语言 | 进入路径 | 来源 |
|------|------|------|---------|------|
| PIE | **\*en**（<span style="color:blue">在…里面</span>） | 原始印欧语 | — | [Wikt-PIE] |
| PIE 扩展 | **\*en-ter**（<span style="color:blue">更里面→在…之间</span>） | 原始印欧语 | → inter-, intra-, enter | [Etym] |
| 古典拉丁 | **in-**（<span style="color:blue">在…里面、进入、朝向</span>） | 拉丁语 | — | [Etym] |
| 古典拉丁 | **im-/il-/ir-**（同化变体） | 拉丁语 | — | [Etym] |
| 古法语 | **en-/em-**（<span style="color:blue">拉丁 in- 的常规音变</span>） | 古法语 | 诺曼征服后 | [Etym] |
| 中古英语 | en-/in- 混用（enquire/inquire, ensure/insure） | 中古英语 | 法语/拉丁双线进入 | [Etym] |
| 现代英语 | in- 确定为默认形式，en- 残留在双胞胎词中 | 现代英语 | 拉丁拼写胜出 | [Etym] |
| 古希腊 | **en**（<span style="color:blue">在…里面</span>）→ en-/em- 借词 | 古希腊语 | enthusiasm, embryo, endemic | [Etym] |

### in-（否定）演化链

| 时期 | 形式 | 语言 | 进入路径 | 来源 |
|------|------|------|---------|------|
| PIE | **\*ne**（<span style="color:blue">不</span>） | 原始印欧语 | — | [Wikt-PIE] |
| PIE 零级 | **\*n̥-**（<span style="color:blue">不</span>，元音消失后的鼻音独立成音节） | 原始印欧语 | — | [Wikt-PIE] |
| 原始日耳曼 | **\*un-**（<span style="color:blue">不</span>） | 原始日耳曼语 | → 英语 un- | [Etym] |
| 古典拉丁 | **in-**（<span style="color:blue">不</span>）+ 同化变体 im-/il-/ir- | 拉丁语 | 文艺复兴后大量直借 | [Etym] |
| 古希腊 | **a-/an-**（<span style="color:blue">不</span>，alpha privative） | 古希腊语 | amoral, anarchy, apathy | [Etym] |
| 现代英语 | in- 否定能力衰减——不再能产，新否定词用 un- 或 non- | 现代英语 | — | [Etym] |

<span style="color:red">关键区别：</span>in-（向内）走了四条路径进入英语（直借拉丁 + 经法语 + 经希腊 + 日耳曼本土 in），in-（否定）走了两条（直借拉丁 + 日耳曼本土 un-）。

## 模块 3：语义锚点 + 知识挂点

### 知识挂点

- 你已经在用 **import**（进口）和 **export**（出口）这对反义词——im- = 向内，ex- = 向外。同一个词根 portare（运）。在 [[词缀/ex-词汇教练]] 里已经练过 export，现在反过来看它的镜像 import。
- in-（向内）和 en-（使…）来自同一个拉丁 in-，只是走的路不同。拉丁 in- 进入法语后 i 降低变成 e，于是有了 en-。所以 import（运进来）和 employ（把人放进去用→雇用）底层的 im-/em- 是同一个东西，一个从拉丁直借，一个经法语转手。
- **un-** 和 **in-** 是同一个 PIE 词根——这是最大的知识挂点。unhappy 和 impossible 的否定部分来自同一个祖形 *ne。一个是日耳曼线，一个是拉丁线。

### 底层核心逻辑

<span style="color:red">in-（向内）：</span>所有现代义，回到同一个空间经验——<span style="color:blue">「从外部，跨过边界，到达内部」</span>。

<span style="color:red">物理进入</span>（invade 走进来、import 运进来、inject 注射进来）→ <span style="color:red">视线/注意力进入</span>（inspect 往里面看、inquire 往里面问、indicate 往里指出）→ <span style="color:red">状态变化——把某物"做进"某状态</span>（inflate 吹气进去→膨胀、intense 拉进去→绷紧→强烈、illuminate 光照进去→照亮→启发）→ <span style="color:red">抽象包含</span>（include 关在里面→包含、involve 卷进去→涉及）。

<span style="color:red">in-（否定）：</span>逻辑简单——<span style="color:blue">「把后面的形容词/名词取反」</span>。但语义剖面不简单：

in- 否定和 un- 否定不一样。un- 倾向于 <span style="color:red">「相反的动作/品质」</span>（undo 是 do 的反转，unkind 是 kind 的反面）。in- 倾向于 <span style="color:red">「缺乏/没有」</span>（inactive 就是没有 activity，不是 activity 的反面——没有一个叫「inact」的动作）。

这是 in- 主要只接形容词/抽象名词而几乎不接动词的原因——它表示「不具备这种属性」，不是「反向做这个动作」。

### 易误解盲区

<span style="color:red">盲区一：in- 双胞胎。</span>同一个 im-/in-/il-/ir- 可能是「向内」也可能是「不」——你必须看第二个成分：

- import = in-（向内）+ portare（运）= 运进来 ✓
- impossible = in-（不）+ possibilis（可能的）= 不可能的 ✓
- <span style="color:red">inflammable = in-（向内！）+ flamma（火焰）+ able = 能烧着的！不是「不能烧的」！</span>

快速鉴别法——两条规则配合使用，准确率远高于单条：
1. <span style="color:red">看词性</span>：否定 in- 几乎只接形容词/抽象名词（inactive, inability）。向内 in- 主要接动词词根构成的动词（invade, include）。看到形容词大概率是否定，看到动词大概率是向内。
2. <span style="color:red">看剩余部分</span>：如果去掉 in- 剩下的不是一个英语单词（\*vade, \*clude, \*ject），那几乎是向内。但如果剩下的恰好是一个词（form → inform，flame → inflame，port → import），**这条规则失灵**——只能靠词性来判。

<span style="color:red">两条规则交叉验证。但以下反例必须记住：</span>

- **形容词 ≠ 铁定否定**：intense、intent、innate、inherent、intimate 都是形容词，但 in- 全是「在里面/向内」——intense = 拉紧在里面→强烈的，不是「不 tense」。
- **剩余部分是单词 ≠ 铁定否定**：inform（form 是单词）、inflame（flame 是单词）、import（port 是单词）、inhabit（habit 是单词）、intense（tense 是单词）——in- 全是「向内」，不是否定。

反例的共同规律：in-（向内）碰到的词根恰好也是一个独立英语单词时，两条规则都失灵。这时候只能靠语义判断——「inform = 给 shape→通知」和「infirm = 不牢固→虚弱的」长得像，但一个向内、一个否定，你只能查词源。

<span style="color:red">盲区二：unable / inability 错配。</span>为什么不是 \*inable 或 \*unability？因为名词 inability 是直接从拉丁借来的完整词（in- + habilitas），而形容词 unable 是英语本土造的（un- + able）。英语借了拉丁名词但自己造了形容词——前缀规则被词源割裂了。同类的还有：unstable / instability，unequal / inequality，ungrateful / ingratitude。

## 模块 4：高频词汇拆解

### A 组：in- 向内—动词词根派生

**1. include** v. <span style="color:blue">包含；包括；把…列为一部分</span>

- 构词：in-（<span style="color:blue">向内</span>）+ claudere（<span style="color:blue">关</span>）
- 字面义：<span style="color:red">关在里面→纳入</span>
- 实际义：①包含、包括（A includes B——B 是 A 的组成部分）②把…列为一部分（include sb in the team）
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| include + 名词/动名词 | <span style="color:blue">包括…</span> |
| be included in + 范围 | <span style="color:blue">被包含在…之内</span> |

- vs contain：<span style="color:blue">都作「包含」解</span>。但是词根不一样：include 的 claudere 意为 to close（关），构词 in- + claudere = 关在里面→纳为一部分；contain 的 tenere 意为 to hold（拿住），构词 con- + tenere = 一起拿住→装在里面。<span style="color:red">所以，include 偏重于把某物列为整体的一部分——强调归属关系；而 contain 侧重于容器把东西装在里面——强调空间容纳。</span>

---

**2. involve** v. <span style="color:blue">涉及；牵扯；包含；使参与；需要</span>

- 构词：in-（<span style="color:blue">向内</span>）+ volvere（<span style="color:blue">卷、滚</span>）
- 字面义：<span style="color:red">卷进去</span>
- 实际义：①涉及、牵扯（The accident involved three cars）②包含、需要（The job involves travel）③使参与（involve sb in decision-making）
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| be involved in + 事情 | <span style="color:blue">卷入/参与…</span> |
| involve + 动名词 | <span style="color:blue">需要做…</span> |

---

**3. insist** v. <span style="color:blue">坚持（观点）；坚称；执意要求</span>

- 构词：in-（<span style="color:blue">向内、在上面</span>）+ sistere（<span style="color:blue">站立</span>）
- 字面义：<span style="color:red">站在里面不走→坚持不动</span>
- 实际义：①坚持观点、坚称（He insisted that he was right）②执意要求（insist on seeing the manager）
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| insist on + 名词/动名词 | <span style="color:blue">坚持要求…</span> |
| insist that + 从句 | <span style="color:blue">坚称…</span> |

- vs persist：<span style="color:blue">都作「坚持」解</span>。但是词根一样（sistere = 站立），前缀不同：insist = in-（在上面）+ sistere = 站在上面不挪开→坚持立场；persist = per-（穿透到底）+ sistere = 站到底→持续存在/坚持行动。<span style="color:red">所以，insist 偏重于坚持一个说法或要求——嘴上的坚持；而 persist 侧重于持续做一件事或持续存在——行动上或时间上的坚持。</span>

---

**4. inspect** v. <span style="color:blue">检查；审视；视察</span>

- 构词：in-（<span style="color:blue">向内</span>）+ specere（<span style="color:blue">看</span>）
- 字面义：<span style="color:red">往里看→仔细看</span>
- 实际义：①仔细检查（inspect the goods）②视察（inspect the troops）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| inspect + 物品/场所 | <span style="color:blue">检查…</span> |

- vs examine：<span style="color:blue">都作「检查」解</span>。但是词根不一样：inspect 的 specere 意为 to look（看），构词 in- + specere = 往里看→目视检查；examine 的 agere 意为 to drive, to weigh（驱使、称重），构词 ex- + agere = 往外称→仔细考量。<span style="color:red">所以，inspect 偏重于用眼睛仔细看——外观、结构、是否有损坏；而 examine 侧重于全面考量——可以看、可以测试、可以提问。</span>

---

**5. inspire** v. <span style="color:blue">激励；启发（想法、作品）；吸入（空气）</span>

- 构词：in-（<span style="color:blue">向内</span>）+ spirare（<span style="color:blue">呼吸</span>）
- 字面义：<span style="color:red">把气吹进去→注入灵魂</span>
- 实际义：①激励、鼓舞（inspire the team）②启发、赋予灵感（The story inspired a film）③吸入（inspire air——生理义，与 expire 呼气相对）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| inspire + 人 + to do | <span style="color:blue">激励某人做某事</span> |
| be inspired by + 事物 | <span style="color:blue">受…启发</span> |

- vs motivate：<span style="color:blue">都作「激励」解</span>。但是词根不一样：inspire 的 spirare 意为 to breathe（呼吸），构词 in- + spirare = 把气吹进去→从内部注入精神；motivate 的 movere 意为 to move（移动），构词 motivus（能移动的）+ -ate = 使移动→从外部推动。<span style="color:red">所以，inspire 偏重于从内心唤起热情——像吸了一口气一样自然涌出；而 motivate 侧重于从外部给人一个理由或动力——推着人往前走。</span>

---

**6. impose** v. <span style="color:blue">强加（规则、观点）；征收（税、罚款）；利用（别人好意）</span>

- 构词：im-（<span style="color:blue">向内→在上面</span>）+ ponere（<span style="color:blue">放置</span>）
- 字面义：<span style="color:red">放在上面→强加</span>
- 实际义：①强加、把…硬塞给（impose rules on sb）②征收（impose a tax）③利用别人的好意（impose on sb's kindness）
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| impose + 规则/限制 + on + 人 | <span style="color:blue">把…强加给…</span> |
| impose + 税/罚款 | <span style="color:blue">征收…</span> |
| impose on + 人 | <span style="color:blue">打扰/利用某人</span> |

---

**7. invade** v. <span style="color:blue">入侵；侵略；大批涌入；侵扰（隐私）</span>

- 构词：in-（<span style="color:blue">向内</span>）+ vadere（<span style="color:blue">走</span>）
- 字面义：<span style="color:red">走进来→闯入</span>
- 实际义：①入侵、侵略（invade a country）②大批涌入（Tourists invade the beach every summer）③侵扰、干涉（invade sb's privacy）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| invade + 国家/领土 | <span style="color:blue">入侵…</span> |
| invade + sb's privacy | <span style="color:blue">侵犯隐私</span> |

- vs intrude：<span style="color:blue">都作「闯入」解</span>。但是词根不一样：invade 的 vadere 意为 to go（走），构词 in- + vadere = 走进来→武力或大量进入；intrude 的 trudere 意为 to thrust（推），构词 in- + trudere = 推进来→不请自来。<span style="color:red">所以，invade 偏重于武力入侵或像潮水一样涌入——规模大、有攻击性；而 intrude 侧重于未经允许挤进来——个体行为、打扰感强。</span>

---

**8. intense** adj. <span style="color:blue">强烈的；紧张的；极其认真的</span>

- 构词：in-（<span style="color:blue">向内</span>）+ tendere（<span style="color:blue">伸展、拉紧</span>）
- 字面义：<span style="color:red">往里拉紧→绷紧的→强烈的</span>
- 实际义：①强烈的、极度的（intense heat/pain）②紧张的、激烈的（intense competition）③极其认真专注的（an intense person）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| intense + 感受/状态 | <span style="color:blue">强烈的…</span> |

- vs fierce：<span style="color:blue">都作「强烈的」解</span>。但是词根不一样：intense 的 tendere 意为 to stretch（拉紧），构词 in- + tendere = 往里面拉紧→内部张力绷到极致；fierce 的 ferus 意为 wild（野的），构词 ferus → fierce = 野兽般的→外部攻击性强。<span style="color:red">所以，intense 偏重于内在张力和集中度——可以用于 heat、emotion、personality；而 fierce 侧重于外在的攻击性和凶猛——用于 competition、animal、battle。</span>

**9. inquire** v. <span style="color:blue">询问；打听；调查</span>

- 构词：in-（<span style="color:blue">向内</span>）+ quaerere（<span style="color:blue">寻求、问</span>）
- 字面义：<span style="color:red">往里探问</span>
- 实际义：①询问、打听（inquire about the price）②调查（inquire into the matter）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| inquire about + 事物 | <span style="color:blue">打听…</span> |
| inquire into + 事件 | <span style="color:blue">调查…</span> |
| inquire whether/why | <span style="color:blue">询问是否/为什么</span> |

### B 组：in- 否定—形容词派生

**10. impossible** adj. <span style="color:blue">不可能的；极难对付的</span>

- 构词：im-（<span style="color:blue">不</span>）+ possibilis（<span style="color:blue">可能的</span>）
- 字面义：<span style="color:red">不可能的</span>
- 实际义：①不可能的（It's impossible to predict）②极难对付的（an impossible child——管不住的孩子）
- 词频层：第 1 层

---

**11. illegal** adj. <span style="color:blue">非法的；违法的</span>

- 构词：il-（<span style="color:blue">不</span>）+ legalis（<span style="color:blue">合法的</span>）
- 字面义：<span style="color:red">不合法的</span>
- 实际义：违反法律的、非法的
- 词频层：第 1 层

- vs unlawful：<span style="color:blue">都作「违法的」解</span>。但是词根不一样：illegal 的 lex 意为 law（成文法），构词 il- + legalis = 违反成文法律的；unlawful 的 lagu 是日耳曼语的 law（法律，含习惯法、道德法）。<span style="color:red">所以，illegal 偏重于违反具体的、成文的法规——可以翻法条找；而 unlawful 侧重于违反任何法律或道德准则——范围更宽、更古老。</span>

---

**12. irregular** adj. <span style="color:blue">不规则的；不正常的；非正规的</span>

- 构词：ir-（<span style="color:blue">不</span>）+ regularis（<span style="color:blue">规则的</span>）
- 字面义：<span style="color:red">不规则的</span>
- 实际义：①不规则的（irregular heartbeat）②不正常的、反常的（irregular behavior）③非正规的（irregular troops——非正规军）
- 词频层：第 2 层

---

**13. inadequate** adj. <span style="color:blue">不充分的；不足的；不够格的</span>

- 构词：in-（<span style="color:blue">不</span>）+ ad-（<span style="color:blue">朝向</span>）+ aequare（<span style="color:blue">使相等</span>）+ -ate
- 字面义：<span style="color:red">没有达到相等水平的→不够的</span>
- 实际义：①不充分的、不足的（inadequate resources）②不够格的、不胜任的（feel inadequate for the job）
- 词频层：第 3 层

- vs insufficient：<span style="color:blue">都作「不足」解</span>。但是词根不一样：inadequate 的 aequare 意为 to make equal（使相等），构词 in- + ad- + aequare = 没有达到应有水平的→质或量上的差距；insufficient 的 facere 意为 to do, to make（做），构词 in- + sub- + facere = 在下面做的→从底下做不够。<span style="color:red">所以，inadequate 偏重于跟某个标准对比——不够好、不够格；而 insufficient 侧重于纯粹数量不够——不够多。</span>

---

**14. inevitable** adj. <span style="color:blue">不可避免的；必然发生的</span>

- 构词：in-（<span style="color:blue">不</span>）+ e-（=ex- <span style="color:blue">向外</span>）+ vitare（<span style="color:blue">避免</span>）+ -able
- 字面义：<span style="color:red">无法从中逃出去的→不可避免的</span>
- 实际义：不可避免的、必然发生的
- 词频层：第 3 层

- vs unavoidable：<span style="color:blue">都作「不可避免」解</span>。但是词根不一样：inevitable 的 vitare 意为 to shun, to avoid（避开），构词 in- + e(x)- + vitare + -able = 无法从里面逃出去的→逻辑上躲不掉；unavoidable 的 void 意为 empty（空），构词 un- + a- + void + -able = 不能让路变成空的→事实上绕不开。<span style="color:red">所以，inevitable 偏重于逻辑或命运上的必然——该来的总会来；而 unavoidable 侧重于现实层面躲不过——没有绕路的选择。</span>

---

**15. independent** adj. <span style="color:blue">独立的；自主的；（国家）独立的；不相关的</span>

- 构词：in-（<span style="color:blue">不</span>）+ de-（<span style="color:blue">向下</span>）+ pendere（<span style="color:blue">悬挂、依赖</span>）+ -ent
- 字面义：<span style="color:red">不挂在别人身上的→不依赖的</span>
- 实际义：①独立的、自主的（independent thinking）②（国家）独立的③不相关的（independent of——独立于…之外、不受…影响）
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| be independent of + 事物 | <span style="color:blue">独立于…、不受…影响</span> |

---

**16. invaluable** adj. <span style="color:blue">极宝贵的；无价的</span>

- 构词：in-（<span style="color:blue">不</span>）+ valuare（<span style="color:blue">估值</span>）+ -able（<span style="color:blue">能被…的</span>）
- 字面义：<span style="color:red">无法估价的→太贵了估不出来</span>
- 实际义：极宝贵的、无价的（= extremely useful/valuable）
- 词频层：第 5 层
- <span style="color:red">⚠️ 陷阱：in- 是否定没错，但否定的是"可估价性"不是"价值"——所以 invaluable = 太贵了估不出来 ≠ 没有价值。</span>

- vs priceless：<span style="color:blue">都作「无价的」解</span>。但是构词不一样：invaluable = in-（不）+ valuable（可估价的）= 无法估价的→太贵了估不出来；priceless = price + -less（没有）= 没有价格的→无法定价的。<span style="color:red">所以，invaluable 偏重于实用价值极高——invaluable advice（无价的建议，因为它太有用了）；而 priceless 侧重于稀有或情感价值——a priceless painting（无价之画，因为它是孤品）。</span>

---

**17. inflammable** adj. <span style="color:blue">易燃的</span>

- 构词：in-（<span style="color:blue">向内！</span>）+ flamma（<span style="color:blue">火焰</span>）+ -able
- 字面义：<span style="color:red">能把火放进去的→能烧着的</span>
- 实际义：易燃的（= flammable）
- 词频层：第 7 层
- <span style="color:red">⚠️ 陷阱之王：这个 in- 是「向内」不是「不」。inflammable = flammable = 能烧着的。反义词是 non-flammable（不能烧的）。</span>

## 模块 5：终极避坑 + 词间连接

### 拼写变化规则

in- 的四个变体，规则是 <span style="color:red">纯语音同化</span>——不涉及 ablaut：

- **in-** → 元音或多数辅音前保持：invade, include, inactive, insane
- **im-** → b/p/m（双唇音）前，n 变成 m：import, impose；impossible, immoral
- **il-** → l 前，n 被 l 完全同化：illuminate；illegal, illiterate
- **ir-** → r 前，n 被 r 完全同化：irrigate；irregular, irresponsible

<span style="color:red">同化逻辑：</span>n 是舌尖齿龈鼻音。在双唇音 b/p/m 前舌头来不及切换——n → m。在 l/r 前 n 被吞掉——n+l → ll，n+r → rr。

### in-（向内）与 ex- 的镜像关系

| | ex- | in-（向内） |
|------|------|------|
| 方向 | <span style="color:blue">向外</span> | <span style="color:blue">向内</span> |
| 变体来源 | 语音同化 | 语音同化（同一种机制） |
| 类型 | 介词/前缀 | 介词/前缀 |
| PIE | \*eghs | \*en |
| 日耳曼同源 | 无 | 有——英语 in 本身就是 |
| classic 反义 | export | import |
| classic 反义 | exclude | include |
| classic 反义 | emigrate | immigrate |

### 名词-形容词前缀错配警示

| 形容词 | 名词 | 错配原因 |
|--------|------|---------|
| unable | inability | 名词借自拉丁（已带 in-），形容词英语自造（带 un-） |
| unstable | instability | 同上 |
| unequal | inequality | 同上 |
| unjust | injustice | 同上 |
| ungrateful | ingratitude | 同上 |
