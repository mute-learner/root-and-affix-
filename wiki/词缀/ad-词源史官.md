## ② 叙事链

### 在 PIE — 源头

PIE **\*ad-**（<span style="color:blue">朝向、在…附近</span>）。小品词——贴到动词前表示动作的方向。它的日耳曼后代至今活在英语里——**at**。at the door = 在门旁边。ad- 和 at 是同源词：一个成了拉丁前缀进入学术词，一个成了英语介词活在日常。[Wikt-PIE]

### 在罗马 — 语义扩张 + 同化大爆发

拉丁语 **ad-**（<span style="color:blue">朝向、往、在…旁边</span>）。语义扩张：空间朝向 → 添加（**addere** = ad- + dare 给 = <span style="color:blue">朝…给→添加</span>）→ 目标指向（**acceptare** = ad- + capere 拿 = <span style="color:blue">朝自己拿→接受</span>）→ 伴随（**assistere** = ad- + sistere 站 = <span style="color:blue">站到旁边→协助</span>）。[Etym]

但 ad- 真正的主角不是语义——是<span style="color:red">同化</span>。拉丁人在日常说话中，ad- 的 -d 被后接辅音吞掉或同化。这不是语法规定的——就是舌头在偷懒：
- -d + c → -c-（**acceptare**）
- -d + f → -f-（**afficere**）
- -d + g → -g-（**aggredi** 攻击）
- -d + l → -l-（**allocare** 分配）
- -d + n → -n-（**annuntiare** 宣布）
- -d + p → -p-（**applicare** 贴合）
- -d + r → -r-（**arripare** 靠岸）
- -d + s → -s-（**assistere** 站旁边）
- -d + t → -t-（**attendere** 伸展过去）

和 con-（同化 -m）、syn-（同化 -n）完全一样的机制。<span style="color:red">ad- 的同化之所以产生 12 种变体而不是 con- 的 5 种，是因为 -d 是舌尖塞音——发音器官比鼻音更灵活，也更容易被后面的辅音完全覆盖。</span>[Etym]

### 在希腊 — 无

PIE \*ad- 在希腊语中无对应前缀。[Wikt-PIE]

### 在日耳曼 — 平行演化：at 的前世今生

PIE \*ad- 进入原始日耳曼语，经过格林定律（\*d → \*t）后变成 **\*at**。这是格林定律的标准操作——PIE 的浊塞音在日耳曼语中清化。古英语继承了这个形式，写作 **æt**，现代英语 **at**。[Etym]

日耳曼线没有把 \*at 发展成前缀系统——这和拉丁线形成了根本性的差异。拉丁人把 ad- 贴到动词前面，造出了成百上千个复合词（accept, affect, arrive……）；日耳曼人保持了 at 的独立介词身份，把它用在名词前面表达空间关系。两种语法策略，同一个 PIE 源头。[Etym]

at 在现代英语中的语义从空间扩展到了时间和抽象域——这是「朝向」的自然延伸：

- **空间**（最早的义层）：at the door = <span style="color:blue">在门旁边</span>，at the station = <span style="color:blue">在车站</span>。at 的空间义恰好对应 ad- 的原始义——朝向一个点并停在那里。
- **时间**（空间→时间的隐喻映射）：at 3pm = <span style="color:blue">在下午三点这个时间点上</span>。把时间想象成一条线，at 标记了线上的一个点——和它在空间中标记一个位置用的是同一个逻辑。
- **抽象方向**：at risk = <span style="color:blue">朝向风险→处于风险中</span>，at best = <span style="color:blue">往最好的方向看→充其量</span>，aim at = <span style="color:blue">瞄向→对准</span>。aim at 尤其能说明问题——动作（aim）的方向由 at 标记，这和拉丁 ad- + 动词的结构在功能上完全平行，只是语法手段不同（拉丁用前缀，日耳曼用介词短语）。

<span style="color:red">at 是英语里使用频率最高的 20 个词之一。</span>你每天说几十遍 at，而 at 和 ad- 六千年前是同一个词。ad- 走了学术路线（accept, admire, adapt），at 走了日常路线（at home, at work, at once）。两个词互不认识，但共享同一个 PIE 祖先。这种「同源词在英语里以不同身份重逢」的模式也出现在 ab-/of 中——日耳曼介词和拉丁前缀在英语中会师，不是巧合，是常态。[Etym]

### 在法国 — 整形手术的第一次：法语怎么削掉了 -d

诺曼征服（1066 年）后，大量拉丁 ad- 词通过法语进入英语。但法语在口耳相传的过程中，对所有 ad- 变体做了一件事：<span style="color:red">把 -d 全部削掉，统一变成 a-。</span>

这不是法语独有的现象——古法语有一个系统性的音变趋势：简化拉丁辅音丛，尤其删除词中不重读的辅音。ad- 的 -d 在法语中不承载重音，又夹在前缀和后接词根之间，成了最容易脱落的目标：

- 拉丁 **acceptare** → 古法语 **accepter**（ac- 保留了 -c，但 ad- 的原始 -d 早已不在——ac- 里的 -c 是来自同化后的形式，不是来自 -d 本身的残留）
- 拉丁 **ad-juxtare**（<span style="color:blue">放到旁边</span>）→ 古法语 **ajuster**（-d 消失，-j- 保留）
- 拉丁 **ad-vantiare**（<span style="color:blue">朝前走</span>，晚期拉丁）→ 古法语 **avancier**（-d 消失 → advance 的前身）
- 拉丁 **ad-venire**（<span style="color:blue">朝…来→到来</span>）→ 古法语 **avenir** → 英语 **advent**（但 advent 是后来从拉丁直借的，拼写恢复了 -d）

到中古英语早期（12-14 世纪），英语里的 ad- 词几乎全部以 a- 形式存在。ajusten, avauncen, aventure——-d 死了。如果历史停在这里，今天的英语就不会有 adjust，只有 ajust；不会有 advance，只有 avance。[Etym]

### 在英国 — 整形手术的第二次：抄写员把 -d 植回去

15 世纪，英国发生了一件语言史上罕见的事——<span style="color:red">抄写员主动把已经死掉的 -d 拼回去。</span>

这不是自然语言演化——这是人为干预。15 世纪的英国正处于文艺复兴早期，古典学术复兴，拉丁语被视为文明语言的标准。当英国抄写员在誊写手稿时，他们看着 ajuster 这样的词，心里知道它的拉丁祖先是 ad-juxtare。于是他们拿起笔，把 -d 加回去了——ajusted → **adjusted**，avance → **advance**，aventure → **adventure**。[Etym]

这场「学术整形」有两个后果：

**大部分恢复是对的。** adjust（ad- + juxta <span style="color:blue">在旁边</span>）、adopt（ad- + optare <span style="color:blue">选择</span>）、adapt（ad- + aptare <span style="color:blue">使适合</span>）——这些词的 -d 确实是拉丁 ad- 的合法后代。恢复拼写只是把被法语削掉的东西重新安回去。

**但有些恢复是错的。** 抄写员有时候过于热心，在本来就与 ad- 无关的词前面也加了 -d：

- **admiral**（<span style="color:blue">海军上将</span>）← 阿拉伯语 **amīr**（<span style="color:blue">指挥官</span>）。和 ad- 毫无关系——是中世纪地中海贸易中从阿拉伯语借来的。抄写员看到 amiral 这个词，觉得 a- 应该是 ad-，就塞了一个 -d 进去。
- **advance**（<span style="color:blue">前进</span>）← 拉丁 **abante**（ab- <span style="color:blue">从</span> + ante <span style="color:blue">前面</span> = <span style="color:blue">从前面</span>）。advance 的真正前缀是 ab-（离开），不是 ad-（朝向）。抄写员误判了。ironically，英语 advance 的法语表亲 avancer 至今没有 -d。

这场整形手术不是 ad- 独有的——同一个时期，英国学者也给 debt 加了 -b（拉丁 debitum）、给 doubt 加了 -b（拉丁 dubitare）、给 island 加了 -s（拉丁 insula——但 island 根本不是从 insula 来的，它来自古英语 īegland）。15 世纪的英国学者对拉丁语有一种近乎宗教般的敬畏，觉得拉丁拼写就是「正确」的拼写。他们的动机是好的，只是偶尔用力过猛。

<span style="color:red">ad- 是现代英语里唯一经历过两次"整形"的前缀——先被法语削掉 -d，再被英国学者植回 -d。你今天每写一个 adjust，那个 d 是 15 世纪一个抄写员从拉丁词典里翻出来、用鹅毛笔帮你加上去的。它不是自然演化出来的——是学术意志的化石。</span>[Etym]

## ③ 音变机制：ad- 的变体逻辑

ad- 不涉及元音变换。变体来自<span style="color:red">拉丁内部的辅音同化</span>——同化的是 -d（con- 同化 -m，syn- 同化 -n）。

| 表面形式 | 触发条件 | 机制 | 例词 |
|---------|---------|------|------|
| **ad-** | 元音、d、h、j、m、v 前 | -d 保持 | adapt, admire |
| **a-** | sc、sp、st 前 | -d 完全脱落 | ascend, aspire |
| **ac-** | c、q 前 | -d → -c | accept, acquire |
| **af-** | f 前 | -d → -f | affect, afford |
| **ag-** | g 前 | -d → -g | aggression |
| **al-** | l 前 | -d → -l | allow |
| **an-** | n 前 | -d → -n | announce |
| **ap-** | p 前 | -d → -p | apply |
| **ar-** | r 前 | -d → -r | arrive |
| **as-** | s 前 | -d → -s | assist |
| **at-** | t 前 | -d → -t | attend |

同化逻辑：-d 是舌尖塞音（舌尖抵齿槽、气流爆破）。在多数辅音前，-d 被后接辅音完全覆盖——变成同一个辅音。在 sc/sp/st 前，-d 在两个擦音/塞音之间无法立足，直接消失。<span style="color:red">con- 同化的是鼻音（-m），syn- 同化的是鼻音（-n），ad- 同化的是塞音（-d）——三种不同的辅音类型，同一种舌头偷懒。</span>

## ④ 跨语族同源词汇总

| 语族 | 语言 | 同源词 | 语义 |
|------|------|-------|------|
| 意大利语族 | 拉丁语 | **ad** | 朝向、往、在…旁边（介词+前缀） |
| 日耳曼语族 | 古英语 | **æt** | 在…旁边、在…时间点 |
| 日耳曼语族 | 现代英语 | **at** | 在（地点/时间/抽象方向） |
| 日耳曼语族 | 古诺斯语 | **at** | 在、朝向 |
| 日耳曼语族 | 哥特语 | **at** | 在、靠近 |
| 日耳曼语族 | 德语 | **az**（古高地德语，已废） | 在、朝向 |
| 凯尔特语族 | 古爱尔兰语 | **ad-** | 朝向（前缀，如 ad-rimi「数、算」） |
| 凯尔特语族 | 威尔士语 | **ad-** | 再、回（语义偏移：朝向→重复） |
| 印度-伊朗语族 | 梵语 | **ádhi** | 在上面、朝向（语义偏移：\*ad- + 后缀 -dhi） |

与 con-（6 个语族）、ab-（6 个语族）相比，ad- 的跨语族分布集中在意大利和日耳曼两条线——希腊线没有，凯尔特线有但语义已偏移，印度-伊朗线加了后缀后语义也有变化。<span style="color:red">ad- 是一个相对"窄"的 PIE 根，但这不影响它在意-日耳曼线内部的巨大产能。</span>[Wikt-PIE][AH]

## ⑤ 学术争议

- **ad- 的 ab- 变体归属**：abbreviate 的 ab- 是 ad- 在 b 前的变体，还是 ab-（离开）前缀？主流视为 ad- 的同化变体——因为 abbreviate 的核心义是「朝短的方向做」而非「从短处离开」。[Etym]
- **中世纪整形手术的范围**：到底有多少词被错加了 ad-？admiral、advance 确定是错加，但 advance 的 ad- 已在现代英语中完全合法化——拼写战胜了词源。
