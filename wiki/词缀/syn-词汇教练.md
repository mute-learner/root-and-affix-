## 模块 1：基础信息

**变体总览**——syn- 在英语里有 <span style="color:red">4 种表面形式</span>，来自 <span style="color:red">1 个 PIE 根</span>。变体由后接辅音决定——和 con- 一样是鼻音同化，但规则不同：con- 同化的是 -m，syn- 同化的是 -n。

| 形式 | 出现条件 | 代表词 |
|------|---------|--------|
| **syn-** | 其余辅音前 | synonym, synthesis, synchronize |
| **sym-** | b、p、m（双唇音）前 | symbol, sympathy, symptom |
| **sy-** | s、z 前 | system |
| **syl-** | l 前 | syllable |

一句话总结：共 4 种表面形式，syn- 是默认形式。全部来自希腊语 σύν（一起）。<span style="color:red">syn- 是 con- 的希腊表亲——同一个语义、不同的 PIE 根、不同的语族路径。</span>

**核心本义**：<span style="color:blue">一起、共同——两个或多个事物同时发生、同时存在、或联合成一个整体</span>

**vs con- 关键区别**：<span style="color:red">syn- 没有「彻底」义。</span>con- 在拉丁语内部发展出强化用法（confirm 彻底加固、conclude 彻底关上），syn- 在希腊语中没有走过这条路。syn- 永远只表示「一起」。

**同源词概览**：
- 希腊线（借词）：英语中绝大多数 syn- 词，文艺复兴后通过学术和科学术语进入
- 拉丁线：对应 con-——<span style="color:red">同语义、不同 PIE 根</span>（syn- < \*ksun-，con- < \*kom）
- 斯拉夫线：俄语 s(o)-——PIE \*ksun- 的另一个后代
- 日耳曼线：<span style="color:red">无</span>——PIE \*ksun- 在日耳曼语族中没有留下常用同源词

**语域层**：
- 学术/科技层：synthesis, synthetic, synchronize, syndrome, symmetry, synapse
- 日常层：system, symbol, sympathy, symptom
- 文化层：symphony, symposium

**词频层**：
- 第 1-2 层核心高频：system, symbol
- 第 3-4 层中高频：sympathy, symptom, synonym, synthesis, synthetic, synchronize, symmetry, syndrome, symphony, syllable, synopsis, symposium, synergy

## 模块 2：词源演化链

syn- 是<span style="color:red">介词/前缀</span>，不是动词词根。和 con-、ex-、in- 同类型。演化路径在希腊语内部完成。

| 时期 | 形式 | 语言 | 进入路径 | 来源 |
|------|------|------|---------|------|
| PIE | **\*ksun-**（<span style="color:blue">和…一起、伴随</span>） | 原始印欧语 | — | [Wikt-PIE] |
| 古希腊 | **σύν (syn)**（<span style="color:blue">一起、共同、同时</span>） | 古希腊语 | — | [Etym] |
| 古希腊 | **συμ- / συλ- / συσ-**（同化变体） | 古希腊语 | — | [Etym] |
| 晚期拉丁 | **syn- / sym-**（希腊借词中介） | 晚期拉丁 | 学术借入 | [Etym] |
| 文艺复兴后 | synthesis, sympathy, system 等 | 早期现代英语 | 希腊→拉丁→英语 | [Etym] |
| 1860 年后 | syn- 高度能产于科学术语 | 现代英语 | 新造词 | [Etym] |

<span style="color:red">syn- 没有拉丁内部的语义创新——希腊人没有把它发展出「彻底」义。</span>这是 syn- 和 con- 最根本的区别：con- = 一起 + 彻底（双义面），syn- = 一起（单义面）。两个前缀在英语中和平共存——拉丁词和希腊词各司其职。

## 模块 3：语义锚点 + 知识挂点

**知识挂点**：
- 你已经认识 system——sy- + histanai（站）→ 站在一起→系统。system 是 syn- 最常用的词。
- 中文母语者：syn- = **同**。同义词 = synonym、同步 = synchronize、同情 = sympathy。syn- 的中文直觉就是「同」——同一个时间、同一个感受、同一个名字。
- con- = 合 + 彻，syn- = 同。拉丁的「合」和希腊的「同」——两个古代文明对「一起」的不同表达。

**底层核心逻辑**（<span style="color:red">红色</span>）：syn- 的所有现代义，回到同一个经验——<span style="color:blue">"两个或多个独立事物在同一时刻、同一空间、或同一关系中共同存在或发生"</span>。

<span style="color:red">共同存在</span>（sympathy 一起感受、symphony 声音一起响、symbiosis 一起生活）→ <span style="color:red">共同构成</span>（synthesis 放在一起→合成、system 站在一起→系统、syllable 拿在一起→音节）→ <span style="color:red">共同发生</span>（synchronize 时间一致→同步、symptom 一起落下→症状、syndrome 一起跑→综合症）→ <span style="color:red">共同对照</span>（symbol 扔在一起→符号——把两半对上、symmetry 一起度量→对称——两边相等）。

<span style="color:red">四个语义层，从「一起感受」到「一起构成」，syn- 始终保持纯「一起」——从未走向「彻底」。</span>

**易误解盲区**：
- ⚠️ **syllabus 不是 syn- 词**——中世纪抄写员把拉丁 sittybas（目录标签）误读成 syllabus。它开头的 syll- 纯属意外。鉴别法：<span style="color:red">拆开看第二部分——syllabus 的 -abus 不是任何希腊词根。真 syn- 词的第二部分都是可辨识的希腊词根（-onym 名、-chron- 时间、-path- 感受）。</span>
- ⚠️ **sym- 不是 sem-**：symmetry（一起+度量）和 seminary（种子+地方）的 sym- 和 sem- 不同源。seminary 里的 semin- 来自拉丁 semen（种子）。
- ⚠️ **syn- vs con- 不互通**：sympathy（一起感受）≠ compassion（一起感受——com- + pati）。同义但不同根——一个希腊一个拉丁。英语里这种「希腊-拉丁双胞胎」很常见。

## 模块 4：高频词汇拆解

从希腊借词中精选 15 个高频词，按变体形式分组。

### syn- 标准形（7 词）

**1. synonym** 名词 <span style="color:blue">同义词</span>

- 构词：syn-（<span style="color:blue">一起</span>）+ onyma（<span style="color:blue">名字</span>）
- 字面义：<span style="color:red">同一个名字——两个词共享一个含义</span>
- 实际义：与另一个词意思相同或几乎相同的词
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| a synonym for/of | ……的同义词 |
| near synonym | 近义词 |

**2. synthesis** 名词 <span style="color:blue">综合；合成</span>

- 构词：syn-（<span style="color:blue">一起</span>）+ thesis（<span style="color:blue">放置</span>）
- 字面义：<span style="color:red">放在一起——把分散的元素组合成一个整体</span>
- 实际义：把不同观点或信息综合在一起形成一个新的整体（综合）；通过化学反应把简单物质合成复杂物质（合成）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| a synthesis of | ……的综合 |
| organic synthesis | 有机合成 |

**3. synthetic** 形容词 <span style="color:blue">合成的；人造的</span>

- 构词：syn-（<span style="color:blue">一起</span>）+ tithenai（<span style="color:blue">放置</span>）+ -tic（<span style="color:blue">形容词后缀</span>）
- 字面义：<span style="color:red">放在一起制成的——不是天然长出来的，是人工拼出来的</span>
- 实际义：用化学方法人工制造的而非天然的（合成的、人造的）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| synthetic materials | 合成材料 |
| synthetic fibers | 合成纤维 |

**4. synchronize** 动词 <span style="color:blue">同步；使同时发生</span>

- 构词：syn-（<span style="color:blue">一起</span>）+ chronos（<span style="color:blue">时间</span>）+ -ize（<span style="color:blue">动词后缀</span>）
- 字面义：<span style="color:red">让时间一致——把两个表调到同一个时刻</span>
- 实际义：让两个或多个设备的数据保持一致（同步数据）；让两个动作在同一时间发生（使同时）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| synchronize with | 与……同步 |
| synchronize data | 同步数据 |

**5. syndrome** 名词 <span style="color:blue">综合症；典型表现</span>

- 构词：syn-（<span style="color:blue">一起</span>）+ dromos（<span style="color:blue">跑、过程</span>）
- 字面义：<span style="color:red">一起跑的症状——一组症状同时出现、结伴而来</span>
- 实际义：一组同时在一个人身上出现的症状，共同构成一种疾病（综合症）；一组典型的行为特征（典型表现）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| Down syndrome | 唐氏综合症 |
| Stockholm syndrome | 斯德哥尔摩综合症 |

**6. synopsis** 名词 <span style="color:blue">概要；大纲</span>

- 构词：syn-（<span style="color:blue">一起</span>）+ opsis（<span style="color:blue">看</span>）
- 字面义：<span style="color:red">一起看——把所有内容一眼看全</span>
- 实际义：把一本书、电影或报告的主要内容压缩成一段简短的概述
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| a synopsis of | ……的概要 |
| plot synopsis | 剧情梗概 |

**7. synergy** 名词 <span style="color:blue">协同效应；1+1>2</span>

- 构词：syn-（<span style="color:blue">一起</span>）+ ergon（<span style="color:blue">工作</span>）
- 字面义：<span style="color:red">一起工作——合力产生比分开更大的效果</span>
- 实际义：两个或多个事物组合在一起产生的效果大于各自单独效果的总和
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| synergy between | ……之间的协同效应 |
| create synergy | 创造协同效应 |

### sym- 双唇音 b/p/m 前（6 词）

**8. symbol** 名词 <span style="color:blue">符号；象征</span>

- 构词：sym-（<span style="color:blue">一起</span>）+ ballein（<span style="color:blue">扔</span>）
- 字面义：<span style="color:red">扔在一起——两半对上→用记号来代表事物</span>
- 实际义：一个形状、图画或字母用来代表某个事物（符号）；一个能唤起某个抽象概念的具体事物（象征）
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| a symbol of | ……的象征 |
| status symbol | 身份象征 |
| mathematical symbol | 数学符号 |

- vs sign：<span style="color:blue">都作「符号」解</span>。symbol = sym-（<span style="color:blue">一起</span>）+ ballein（<span style="color:blue">扔</span>）——「扔在一起」→ 两半对上才能辨识→约定俗成的记号——需要文化和共识才能读懂；sign = 拉丁 signum（<span style="color:blue">标记、记号</span>）——直接的指示标记——不需要文化共识也能理解。<span style="color:red">symbol 偏重于文化约定——同一个 symbol 在不同文化中含义不同（十字架在基督教是救赎、在古罗马是刑具）；sign 侧重于直接指示——有烟就是有火，不需要文化知识。</span>红灯是 sign（直接告诉你停），国旗是 symbol（需要文化背景才能理解其含义）。

**9. sympathy** 名词 <span style="color:blue">同情；赞同</span>

- 构词：sym-（<span style="color:blue">一起</span>）+ pathos（<span style="color:blue">感受、情感</span>）
- 字面义：<span style="color:red">一起感受——别人疼你也在心里疼</span>
- 实际义：对他人的不幸或痛苦感到难过和理解（同情）；对某个观点或立场表示支持（赞同）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| have/feel sympathy for | 同情…… |
| in sympathy with | 赞同…… |
| you have my sympathy | 我同情你 |

**10. symptom** 名词 <span style="color:blue">症状；征兆</span>

- 构词：sym-（<span style="color:blue">一起</span>）+ piptein（<span style="color:blue">落下、发生</span>）
- 字面义：<span style="color:red">一起发生的事——病症不会单独来，是一起来的</span>
- 实际义：身体上表现出疾病的存在（症状）；某个更大问题的外在迹象（征兆）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| symptom of | ……的症状/征兆 |
| show symptoms | 表现出症状 |
| early symptom | 早期症状 |

**11. symmetry** 名词 <span style="color:blue">对称；匀称</span>

- 构词：sym-（<span style="color:blue">一起</span>）+ metron（<span style="color:blue">度量</span>）
- 字面义：<span style="color:red">一起度量——左边和右边是一样的尺寸</span>
- 实际义：物体的两边在形状、大小或排列上完全相同和镜像对应（对称）；面部或身体的均衡美感（匀称）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| perfect symmetry | 完美对称 |
| bilateral symmetry | 两侧对称 |

**12. symphony** 名词 <span style="color:blue">交响乐</span>

- 构词：sym-（<span style="color:blue">一起</span>）+ phone（<span style="color:blue">声音</span>）
- 字面义：<span style="color:red">声音一起响——几十种乐器同时发出声音、和谐地融合在一起</span>
- 实际义：由管弦乐队演奏的大型音乐作品，通常有多个乐章
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| symphony orchestra | 交响乐团 |
| Beethoven's Fifth Symphony | 贝多芬第五交响曲 |

**13. symposium** 名词 <span style="color:blue">研讨会；专题讨论会</span>

- 构词：sym-（<span style="color:blue">一起</span>）+ posis（<span style="color:blue">喝、饮</span>）
- 字面义：<span style="color:red">一起喝酒——古希腊人饭后喝酒聊天讨论哲学</span>
- 实际义：一群专家聚集在一起讨论某个特定主题的正式会议
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |

| hold a symposium | 举办研讨会 |

## 模块 5：终极避坑 + 词间连接

### 拼写变化规则

syn- 的变体来自<span style="color:red">希腊内部的鼻音同化</span>——和 con- 是同一类机制（辅音同化），但同化的鼻音不同：con- 同化的是 -m，syn- 同化的是 -n。

| 表面形式 | 触发条件 | 机制 | 例词 |
|---------|---------|------|------|
| **syn-** | 其余辅音前 | -n 保持 | synonym, synthesis, synchronize |
| **sym-** | b、p、m（双唇音）前 | -n 变成 -m——舌尖鼻音变成双唇鼻音，匹配后续双唇辅音 | symbol, sympathy, symptom |
| **sy-** | s、z 前 | -n 脱落——两个擦音之间鼻音被吞掉 | system |
| **syl-** | l 前 | -n 被 l 完全同化——舌尖鼻音被舌尖流音吞掉 | syllable |

同化逻辑：n 是舌尖鼻音（舌尖抵齿槽）。在双唇音 b/p/m 前舌头要来回切——舌尖 n → 双唇 b → 太麻烦，变成双唇鼻音 m 一气呵成。在 l 前 n 直接被吞——两个都是舌尖音，舌头不想做两遍。在 s/z 前 n 直接消失——擦音空隙里鼻音站不住。

### syn- 与 con- 对比总结

| | syn-（希腊） | con-（拉丁） |
|------|------|------|
| PIE 根 | \*ksun- | \*kom |
| 语义 | 一起（单义面） | 一起 + 彻底（双义面） |
| 变体 | syn-/sym-/sy-/syl- | co-/com-/col-/cor-/con- |
| 同化鼻音 | -n | -m |
| 语域 | 学术/科技/文化 | 日常/学术通用 |
| 例 | sympathy 同情 | compassion 同情（同义双根） |
| 例 | synthesis 合成 | composition 组成（同义双根） |

### 假词根甄别

| 单词 | 误解 | 真相 |
|------|------|------|
| **syllabus** | 以为 syll- 是 syn- 在 l 前的变体 | 中世纪抄写员误读——来自希腊 sittyba（目录标签），与 syn- 无关 |
| **seminary** | 以为 sem- 是 syn- 的变体 | sem- 来自拉丁 semen（种子）——「种子生长的地方」→ 神学院 |
| **symptom ≠ sign** | 以为 symptom 和 sign 一样 | symptom = 身体内在疾病的信号（需要推断），sign = 可直接观察的标记 |

<span style="color:red">快速鉴别法：syn- 词的第二个部分必须是可辨识的希腊词根（-onym 名、-chron- 时间、-path- 感受、-thes- 放置、-metr- 度量、-phon- 声音）。如果不是→查词典。</span>
