## ① 演化全貌

## ② 叙事链

### 在 PIE — 源头：自己的事自己做

六千年前，PIE 人有一个反身代词 **\*s(w)e-**，意思是<span style="color:blue">自己、自身的</span>。这不是空间词——不描述方向、不描述位置。它描述的是归属和身份：这个东西是我自己的，这件事是我自己做的，这个人属于我的族群。

\*s(w)e- 在 PIE 语法体系中的核心用法是反身——指回句子的主语。它衍生出了夺格形式 **\*sē**（<span style="color:blue">靠自己、独自</span>）。夺格在 PIE 中表示来源、工具、方式——\*sē 字面义就是「靠自己」：自己一个人做，不跟别人一起。

<span style="color:red">\*sē → 拉丁 sē- 的关键语义跳变是：「靠自己一个人做自己的事」→「不和别人一起」→「分开、分离」。</span>这个跳变在拉丁语之前就已经完成了——到了古典拉丁时期，sē- 已经是一个成熟的分离前缀，不再带有「自己」的意思。但底层的逻辑始终是：分离的前提是有自己和他人的区分——你得先有「自己」的概念，才能有「把自己和别人分开」的动作。[Wikt-PIE]

\*s(w)e- 在 PIE 中还有一个重要特征：它和「族群」「血缘」概念紧密关联。同源词在日耳曼和印度-伊朗语族中都发展出了「自己的族人」「自己的群体」的语义（sibling = 自己的同胞，Sanskrit sva- = 自己的族群）。这说明 \*s(w)e- 从一开始就不只是个体意义的「自我」——它包含了「属于我的群体」的内外分界。这个内/外分界，是后来 se-「分离」义的真正心理基础：分离 = 区分内和外。[Wikt-PIE]

### 在罗马 — 从「自己」到「分开」：代词的语法化

罗马人继承了 PIE \*sē，发展出拉丁语前缀 **sē-**（<span style="color:blue">分离、分开、放在一边</span>）。此时 sē- 已经完全语法化——罗马人已经不觉得它和反身代词有关系了（反身代词在拉丁语中由 sē/sibi/suī 承担）。sē- 在他们的语言直觉里只是一个表示分离的前缀，和一个空间小品词没有区别。

拉丁 sē- 的语义扩张沿着三条路径展开：

**路径一：物理掰开。** **separare** = sē- + parare（<span style="color:blue">准备、安排</span>）→ 字面义是「各自准备、各自安置」→ 把原来在一起的东西拆开。注意 parare 的核心不是「准备饭」里的准备——在拉丁语中 parare 涵盖「安排、布置、使…就位」——separare 的画面是「把东西分到不同的位置上去各自安顿」。**secernere** = sē- + cernere（<span style="color:blue">筛、分辨</span>）→ 筛出来放到一边 → 由此得出 secretum：被筛到一边藏起来的 → secret。[Etym]

**路径二：挑出甄别。** **seligere** = sē- + legere（<span style="color:blue">收集、挑选</span>）→ 从一堆中挑出来 → select。legere 的本义是「收集」——古罗马人选橄榄就是用手从枝上把果实捋下来收集到一起。se- + legere 是「从收集物中分出来」——先收集全部选项，再从中甄别你要的。这和今天的「单选」逻辑正好相反——不是从零开始挑，而是先把池子建好再往外挑。[Etym]

**路径三：偏离正道。** **sēducere** = sē- + ducere（<span style="color:blue">引导、带领</span>）→ 引到岔路上去 → seduce。**sēditio** = sēd- + ire（<span style="color:blue">走</span>）→ 从当局身边走开 → 煽动叛乱。sēd- 变体在此出现——sē- 在元音开头的词根前（ire 以 i 开头）插入 -d- 避免两个元音碰撞。sedition 不是一个「人走了就完了」的词——它特指用言语煽动他人一起走，所以加上了名词后缀 -io（动作/过程）。[Etym]

**一个特殊的语义：脱离担心。** **sēcurus** = sē- + cura（<span style="color:blue">担心、挂念</span>）→ 把担心放在一边 → 不用挂念的、安心的 → secure。这个语义在拉丁语中就已经高度抽象了——se- 在这里不是物理分离，而是心理分离：把担心的情绪从心里移除。cura 在拉丁语中覆盖面很广——从日常的操心到正式的监护责任——sēcurus 涵盖了所有层次的「放下心来」。[Etym]

### 在希腊 — 另一条路：从「自己」到「私人的」

同一个 PIE \*s(w)e- 进入古希腊语，走了和拉丁完全不同的路。

希腊语中，\*s(w)e- 的后代是 **ἑ-**（he-），没有发展成前缀系统，而是保持了形容词和名词的身份。最关键的后代是 **ἴδιος**（idios = <span style="color:blue">自己的、私人的</span>）——来自 \*swe-d-ios（\*swe- 自己 + 后缀 -d- + -ios）。希腊人的语义方向不是「分离」，而是「属于个人的、不公开的」：

- **idiot** = idios + -tēs（人）→ 字面义是「只关心自己事务的人」→ 在古希腊城邦中，这个词指不参与公共政治的人——在雅典民主制下这是一个严重的贬义词：一个好公民必须参与城邦事务，不参与的人是「私人」。后经由拉丁语 idiota 进入英语，语义从「不参与公共事务的人」→「无知的人」→「傻瓜」。<span style="color:red">idiot 的语义坠落史是雅典民主制的化石——一个城邦的价值观（公民必须参与公共生活）凝结在一个词里。</span>
- **idiom** = idios + -ōma（名词后缀）→ 字面义是「自己特有的东西」→ 一个人或一个地方特有的语言表达方式 → 习语、成语。

希腊线的核心区别在于：拉丁人把 \*s(w)e- 变成了分离前缀（se- = 分开），希腊人把它变成了私属形容词（idios = 自己的）。同一个 PIE 根，「自己」→ 拉丁人强调「把不是自己的分出去」，希腊人强调「保留属于自己的那部分」。[Etym]

### 在日耳曼 — self：se- 失散六千年的双胞胎

这是 se- 故事里最精彩的部分。PIE \*s(w)e- 进入原始日耳曼语，发展出三个后代，每一个都活在现代英语中：

**self**（<span style="color:blue">自己</span>）。原始日耳曼语 **\*selbaz**（< s(w)e- + \*-l- 后缀 + \*-bʰo- 后缀）→ 古英语 **self / seolf** → 现代英语 **self**。self 保持了 PIE 最原始的「自己」义——毫不偏离。和 se- 的语义跳变放在一起看：self = 自己（保持原义），se- = 自己→分开（语义跳变）。<span style="color:red">self 和 se- 六千年前是同一个词——self 守住了老家，se- 搬去了罗马。</span>

**sibling**（<span style="color:blue">兄弟姐妹</span>）。古英语 **sibb**（<span style="color:blue">血缘关系、亲属</span>）+ -ling（小称后缀）→ 字面义是「属于自己的小人儿」→ <span style="color:blue">同胞</span>。sibb 的核心是「血缘归属」——属于同一个血族的人。这和 PIE \*s(w)e- 的族群含义直接相连。现代英语 sibling 在 20 世纪被人类学家复活（之前是一个已死的古英语词）用来做一个性别中立的「兄弟或姐妹」术语——人类学需要它来讨论亲属关系而不必指定性别。sibling 是学术意志复活死词的罕见成功案例。[Etym]

**swain**（<span style="color:blue">乡村少年、求婚者</span>）。古诺斯语 **sveinn**（<span style="color:blue">自己的随从、年轻侍从</span>）→ 中古英语 swein → 现代英语 swain。语义从「自己的侍从」→「年轻男子」→「乡村青年」→ 文学用语中「求婚者」。今天这个词几乎只出现在古典文学中（"a handsome young swain"），但它和 self、sibling 是同源词——都属于 PIE「自己人」的语义圈。[Etym]

日耳曼线对英语的影响远超拉丁借词 sē-。self 是你每天说几十遍的词，sibling 是社会科学的基石术语。而它们和 separate 的 se- 是同一个 PIE 词。你每说一次 "I'll do it myself"，那个 self 就是 se- 在日耳曼家守了六千年的老宅。[Etym]

### 在法国 — separate 和 sever：一个词被撕成两半

诺曼征服后，拉丁 se- 词通过法语大量进入英语。法语中介给 se- 留下的最深刻的痕迹不是拼写变化——是**把一个词撕成了两个**。

拉丁 **separare**（sē- + parare）经过两条路进入英语：

- **直借路线 → separate**（15 世纪）。文艺复兴学者直接从拉丁借入 separare，保留了完整的拼写和-par-部分。语域：通用——科学论文、日常对话、法律文书都能看到 separate。
- **法语路线 → sever**（13 世纪）。se- + parare → 古法语 **sevrer**（parare 的 -par- 被法语磨到只剩 -v-：parare → parer → vrer → vrer，辅音一路脱落）→ 中古英语 **sever**。语域：偏书面——法律术语（sever ties）、医学用语（sever a limb）。

<span style="color:red">同一个拉丁词，穿西装的叫 separate，穿便装的叫 sever。</span>而且语义也发生了分化：separate 保留了「分开」的一般含义（可逆的——分开的东西可以再合起来），sever 发展出了更极端的「切断」含义（不可逆的——一刀两断）。一个词被法语磨过之后，不光是拼写变短了，意思也变狠了。[Etym]

另一个有趣的 doublet：**secure** 和 **sure**。拉丁 sēcurus（se- + cura <span style="color:blue">不用挂念</span>）→ 直借路线 → secure（15 世纪，完整拼写）；→ 法语路线 → OF seur → sure（13 世纪——cura 被磨到只剩 u）。secure = 安全的（更正式、偏书面），sure = 确定的（更日常、偏口语）。同一个拉丁 sēcurus，穿西装的是 secure，便装磨到只剩四个字母——sure。[Etym]

### 在英国 — 三个同源层的会师

文艺复兴时期（15-17 世纪），英国学者批量直借拉丁 se- 词：**select**（seligere）、**seclude**（secludere）、**segregate**（segregare）。这些词没有走法语通道——拼写更接近拉丁原文。

但 se- 在英语里真正的力量不是借词量——而是它让英语有了三个层次的「自己/分离」表达：

| 层 | 形式 | 语义方向 | 代表词 |
|----|------|---------|--------|
| 日耳曼日常层 | **self, sibling** | 自己、自己人——保持 PIE 原义 | myself, yourself, sibling |
| 拉丁分离前缀层 | **se-** | 分开、偏离——从「自己」进化到「分开」 | separate, select, secret |
| 拉丁心理层 | **secure/sure** | 摆脱担心——从「分开担心」到「安心」 | secure, sure |

<span style="color:red">三层的底层是同一个 PIE \*s(w)e-。</span>self 守住了「自己」的原义，se- 走了「自己→单独→分开」的进化路，secure 走了「分开担心→安心」的平行进化路。你用 myself 思考自我，用 separate 分类事物，用 secure 确认安全——三个动作，六千年前是同一张嘴说出的同一个词。

se- 在现代英语中的语义格局：物理掰开（separate、sever、segregate）→ 甄别选出（select、secret）→ 偏离正道（seduce、sedition）→ 脱离世俗（seclude）→ 摆脱挂念（secure）。<span style="color:red">从「自己是自己的」到「我和你掰了」——se- 用了六千年走完了这条从自我意识到社会关系的路。</span>[Etym]

## ③ 音变机制：se- 的无变体与 sed- 插音

se- <span style="color:red">没有辅音同化变体</span>——这在英语前缀系统中极为罕见（另一个是无变体前缀是 pro- 和 pre- 等）。原因很简单：se- 只有一个辅音 s，而 s 是擦音，几乎可以和任何后面的辅音共存——不需要像 -d（ad-）、-m（con-）、-n（syn-）那样调整自己。s 是一个随和的音。

唯一的特殊情况是 **sēd-**——se- 在元音开头的词根前插入 -d-：

| 形式 | 条件 | 机制 | 例词 |
|------|------|------|------|
| **se-** | 辅音前（默认） | s- + 辅音——直接拼合 | separate, select, secret |
| **sēd-** | 元音前 | 插入 -d- 避免两个元音粘连：sē-ire → sēd-itio | sedition |
| **sēd-** | u 前（半元音） | sē-ducere → seduce | seduce |

sēd- 的 -d- 是一个「插音」（hiatus-breaking consonant）——拉丁语极度避免两个元音直接相邻（hiatus 在拉丁诗歌中是一个需要避开的节奏瑕疵）。在 sē- + ire 之间插入一个舌尖塞音 d，让气流在 s 和 i 之间有一个闭合爆破过渡。这和英语中 "a apple" → "an apple"（插入 n）是同一个机制——舌头需要一个辅音来隔开两个元音。[Etym]

<span style="color:red">sēd- 不是 se- 的变体，是 se- 穿了一件防撞衣。</span>

## ⑤ 学术争议

- **\*s(w)e- 中的 w 的状态**：PIE \*s(w)e- 中的 w 是否是词根本身的一部分，还是仅仅在某些词形中出现的过渡音？传统重建（Pokorny）写了 \*se- / \*swe- 两种形式，暗示 w 是随语境变化的可选成分。Rix（LIV²）倾向于将 \*swe- 视为基本形式，\*se- 为弱化变体。拉丁 sē- 来自没有 w 的弱化形式（\*sē），日耳曼 \*selbaz 中的 -l- 可能间接反映了 w 的痕迹（\*swe-l- → \*sel-）。对学习者来说：记住 se- = self 是同源就够了——w 的有无是历史语言学家的战场。[Wikt-PIE]
- **拉丁 sēd- 的 d 来源**：主流认为 sēd- 的 -d 是插音（避免 hiatus）。但有一个少数派假说认为 sēd- 可能来自 PIE 另一个小品词 \*dʰe（放置、做）的遗留，sē- + dʰe = 「单独放置→分开」。这个假说缺乏平行证据（其他前缀如 ad- 在元音前不加 -d），主流学界不接受。但偶尔会在老旧的词源字典中看到这个说法。[Wikt-Lat]
- **se- 在现代英语中的能产性**：se- 在现代英语中 <span style="color:red">完全不能产</span>——没有人用 se- 造新词。可能的原因是 se- 的衍生词几乎全是动词（separate, select, seduce, segregate），但英语的动词新造词现在几乎不用拉丁前缀了（不像 co- 可以随意加到 coworker 上）。se- 是一个封闭的类——你只能使用已经存在的 se- 词，但不能再造新的。
