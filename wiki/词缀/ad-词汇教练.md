## 模块 1：基础信息

**变体总览**——ad- 在英语里有 <span style="color:red">12+ 种表面形式</span>，是整个前缀系统里最复杂的单项。变体来自拉丁内部的辅音同化——ad- 的 -d 被后接辅音吞掉或同化。

| 形式 | 出现条件 | 代表词 |
|------|---------|--------|
| **ad-** | 元音、d、h、j、m、v 前 | adapt, adopt, admire, adjust |
| **a-** | sc、sp、st 前 | ascend, aspire |
| **ab-** | b 前 | abbreviate |
| **ac-** | c、q 前 | accept, acquire |
| **af-** | f 前 | affect, afford |
| **ag-** | g 前 | aggression |
| **al-** | l 前 | allow |
| **an-** | n 前 | announce |
| **ap-** | p 前 | apply, appoint |
| **ar-** | r 前 | arrive |
| **as-** | s 前 | assist, associate |
| **at-** | t 前 | attend, attract |

一句话总结：共 12+ 种表面形式，核心只有 ad- 一个——-d 被后接辅音吞掉或同化。<span style="color:red">ad- 的同化规则和 con- 本质上一样——都是舌头偷懒。</span>

**核心本义**：<span style="color:blue">朝向、往……方向去——一种向某个目标靠近的运动或倾向</span>

**同源词概览**：
- 拉丁线（借词）：英语中绝大多数 ad- 词
- 日耳曼线：古英语 æt（在……）→ 现代英语 at——<span style="color:red">ad- 的本土表亲</span>。at 和 ad- 是同源词，一个是独立介词，一个是前缀
- 希腊线：无直接同源

**语域层**：
- 拉丁学术层：acquire, attribute, aggression, adapt, advocate
- 日常层：accept, arrive, attend, allow, apply, afford, adjust

**词频层**：
- 第 1 层核心高频：accept, arrive, allow, apply
- 第 2-3 层高频：attend, attract, affect, afford, announce, assist, associate, acquire, adapt, adopt, adjust

## 模块 2：词源演化链

ad- 是<span style="color:red">介词/前缀</span>，不是动词词根。演化路径在拉丁语内部完成。

| 时期 | 形式 | 语言 | 进入路径 | 来源 |
|------|------|------|---------|------|
| PIE | **\*ad-**（<span style="color:blue">朝向、在…附近</span>） | 原始印欧语 | — | [Wikt-PIE] |
| 古典拉丁 | **ad-**（<span style="color:blue">朝向、往、在…旁边</span>） | 拉丁语 | — | [Etym] |
| 古典拉丁 | **ac-/af-/ag-/al-/an-/ap-/ar-/as-/at-** 等（同化变体） | 拉丁语 | — | [Etym] |
| 古法语 | **a-**（所有变体统一弱化） | 古法语 | 诺曼征服后 | [Etym] |
| 中古英语 | **ad- 被"纠正"回来**——抄写员按拉丁拼写恢复了 -d | 中古英语 | 15 世纪学术纠正 | [Etym] |
| 现代英语 | **ad- + 各同化变体** 共存 | 现代英语 | — | [Etym] |

<span style="color:red">ad- 经历过一次"整形手术"——古法语把所有变体都简化成了 a-，15 世纪英国抄写员又按拉丁原文把 -d 拼回去。</span>adjust 如果不整形应该是 ajust，advance 应该是 avance。这场学术纠正留下了一个有趣的副作用：有些词被错加了 ad-（如 admiral 来自阿拉伯语 amīr，与 ad- 无关）。

## 模块 3：语义锚点 + 知识挂点

**知识挂点**：
- 中文母语者：ad- = **往**。往自己这边拿 = accept，往岸靠 = arrive，往…拉 = attract。ad- 的空间直觉就是中文的「往」——指向一个方向并朝它移动。
- 你已经认识 at——古英语 æt，和 ad- 是同源词。at the door = 在门旁边。ad- 和 at 共享 PIE \*ad-，一个成了拉丁前缀，一个成了英语介词。
- ex- = 出，in- = 入，ad- = 往。三个方向前缀：出、入、往。

**底层核心逻辑**（<span style="color:red">红色</span>）：ad- 的所有现代义，回到同一个经验——<span style="color:blue">"从自己所在的位置出发，朝某个目标移动或用力"</span>。

<span style="color:red">物理靠近</span>（arrive 靠岸→到达，attend 朝…伸展→到场，aggression 朝…走→攻击）→ <span style="color:red">力量指向</span>（attract 朝自己拉→吸引，affect 朝…做→施加影响，apply 朝…折叠→贴合）→ <span style="color:red">心理/社会指向</span>（accept 朝自己拿→接受，admire 朝…惊叹→钦佩，advocate 朝…呼喊→声援）→ <span style="color:red">添加增补</span>（add 加，adopt 拿过来→采用）。

四个语义层，同一个方向动作。

**易误解盲区**：
- ⚠️ **ad- 被中世纪抄写员错加**：admiral（阿拉伯语 amīr——"指挥官"）、advance（拉丁 abante——"从前面"）的 ad- 是错加的。<span style="color:red">好在这几个词极少，遇到疑似 ad- 词但拆不出第二部分时，查词典。</span>
- ⚠️ **a- 的多重身份**：ascend 的 a- 是 ad-（sc 前脱落），amoral 的 a- 是希腊否定前缀（"无"），awake 的 a- 是古英语 on。同一个 a-，三个不同来源。鉴别法：<span style="color:red">a- + 动词/形容词 → 看第二部分——希腊词根（moral）→ 否定；拉丁词根（scend）→ ad- 变体。</span>

## 模块 4：高频词汇拆解

从 ad- 衍生词中精选 15 个高频词。按变体形式分为九组。

### ac- c/q 前（2 词）

ad- 在 c 前，-d 被 /k/ 音同化为 -c。

**1. accept** 动词 <span style="color:blue">接受；承认；同意</span>

- 构词：ac-（<span style="color:blue">朝向</span>）+ capere（<span style="color:blue">拿、取</span>）
- 字面义：<span style="color:red">朝自己拿过来——对方递来的东西你接住了</span>
- 实际义：收下别人给予的东西或提议（接受）；承认某事是真实的或正确的（承认、认可）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| accept an offer | 接受提议 |
| accept responsibility | 承担责任 |
| accept that | 承认…… |

- vs receive：<span style="color:blue">都作「收到」解</span>。accept = ac-（<span style="color:blue">朝向</span>）+ capere（<span style="color:blue">拿</span>）——「朝自己拿」→ 主动伸出手去接——有主观意愿的接纳；receive = re-（<span style="color:blue">回</span>）+ capere（<span style="color:blue">拿</span>）——「拿回来」→ 被动地收到——东西来了你就收到了。<span style="color:red">accept 偏重于主观接受——你愿意接收（accept a gift 你愿意收下这个礼物）；receive 侧重于客观收到——东西进了你的范围（receive a letter 信到了你手里，不管你想不想看）。</span>

**2. acquire** 动词 <span style="color:blue">获得；习得；收购</span>

- 构词：ac-（<span style="color:blue">朝向</span>）+ quaerere（<span style="color:blue">寻求、获取</span>）
- 字面义：<span style="color:red">朝目标寻求直到拿到手——主动去找然后得到</span>
- 实际义：通过自己的努力或购买获得某物（获得、习得）；公司通过购买获得另一家公司（收购）
- 词频层：第 3 层
- 搭配：

| 搭配模式                | 说明     |     |
| ------------------- | ------ | --- |
| acquire knowledge   | 获取知识   |     |
| acquire a company   | 收购公司   |     |
| acquire a taste for | 开始喜欢…… |     |

**3. arrive** 动词 <span style="color:blue">到达；来临；出生</span>

- 构词：ar-（<span style="color:blue">朝向</span>）+ ripa（<span style="color:blue">河岸、海岸</span>）
- 字面义：<span style="color:red">靠到岸上——船经过长途航行终于触到了陆地</span>
- 实际义：经过一段旅程后到达目的地（到达）；某个时间点或事件来到（来临）
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| arrive at/in + 地点 | 到达某地 |
| arrive at a conclusion | 得出结论 |
| newly arrived | 新来的 |

- vs reach：<span style="color:blue">都作「到达」解</span>。arrive = ar-（<span style="color:blue">朝向</span>）+ ripa（<span style="color:blue">岸</span>）——「靠岸」→ 旅程的终点——强调到达这个结果，不及物（arrive at the station）；reach = 古英语 rǣcan（<span style="color:blue">伸出、够到</span>）——「把手伸出去触到」→ 强调够到的动作，及物（reach the station）。<span style="color:red">arrive 偏重于到达的结果——一段旅程结束了；reach 侧重于够到或抵达的动作——可能还在路上、刚触到。</span>

### ad- 标准形（4 词）

ad- 在元音、d、h、j、m、v 前保持完整。

**6. adjust** 动词 <span style="color:blue">调整；适应</span>

- 构词：ad-（<span style="color:blue">朝向</span>）+ juxta（<span style="color:blue">在旁边、靠近</span>）
- 字面义：<span style="color:red">放到旁边——把一个东西挪到和另一个东西对齐的位置</span>
- 实际义：对某物做小的改变使其更准确或更有效（调整）；慢慢习惯新环境（适应——adjust to）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| adjust the volume | 调整音量 |
| adjust to + 新环境 | 适应…… |
| adjust your expectations | 调整你的预期 |

- vs adapt：<span style="color:blue">都作「适应」解</span>。adjust = ad-（<span style="color:blue">朝向</span>）+ juxta（<span style="color:blue">旁边</span>）——「放到旁边对齐」→ 做小幅微调——小的、可逆的改动；adapt = ad-（<span style="color:blue">朝向</span>）+ aptare（<span style="color:blue">适合</span>）——「朝适合的方向改」→ 做大的、结构性的改变——改到适合新环境为止。<span style="color:red">adjust 偏重于微调校准——在现有基础上拧一拧（adjust the seat 调座位）；adapt 侧重于深度改造——为适应新环境而改变自身（adapt the novel for film 把小说改编成电影）。</span>

**7. admire** 动词 <span style="color:blue">钦佩；欣赏</span>

- 构词：ad-（<span style="color:blue">朝向</span>）+ mirari（<span style="color:blue">惊奇、惊叹</span>）
- 字面义：<span style="color:red">朝……惊叹——看着某物发出赞叹</span>
- 实际义：对某人或某物的品质或成就感到极大的尊重和欣赏
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| admire sb for sth | 因……钦佩某人 |
| I admire your courage | 我钦佩你的勇气 |
| stand back and admire | 退后一步欣赏 |

**8. adapt** 动词 <span style="color:blue">适应；改编</span>

- 构词：ad-（<span style="color:blue">朝向</span>）+ aptare（<span style="color:blue">使适合、调整</span>）
- 字面义：<span style="color:red">朝适合的方向改——把自己调到和新环境匹配</span>
- 实际义：改变自己的行为或结构以适应新环境（适应）；把一个作品从一种形式改到另一种形式（改编——adapt a novel for film）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| adapt to + 新环境 | 适应…… |
| adapt a novel for film | 把小说改编成电影 |
| adapt and survive | 适应并生存 |

- vs adjust：<span style="color:red">见 adjust。</span>

**9. adopt** 动词 <span style="color:blue">采用；收养；正式通过</span>

- 构词：ad-（<span style="color:blue">朝向</span>）+ optare（<span style="color:blue">选择、希望</span>）
- 字面义：<span style="color:red">朝……选择——把别人的东西选过来变成自己的</span>
- 实际义：从法律上把别人的孩子收为自己的孩子（收养）；开始使用某种方法或策略（采用）；正式投票通过法律或政策（正式通过）

**10. affect** 动词 <span style="color:blue">影响；感动</span>

- 构词：af-（<span style="color:blue">朝向</span>）+ facere（<span style="color:blue">做</span>）
- 字面义：<span style="color:red">朝……做——对某物施加一个动作，改变它的状态</span>
- 实际义：对某人或某事物产生影响或使其发生变化（影响）；在情感上打动某人（感动）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| affect + 名词 | 影响…… |
| be deeply affected | 深受感动/影响 |
| adversely affect | 对……产生不利影响 |

- vs effect：<span style="color:blue">都作「影响」解</span>——这是英语中最经典的近义词混淆之一。affect = af-（<span style="color:blue">朝向</span>）+ facere（<span style="color:blue">做</span>）——「朝……做」→ 施加影响的过程——动词为主；effect = ef-（<span style="color:blue">彻底</span>）+ facere（<span style="color:blue">做</span>）——「彻底做出来」→ 影响产生的结果——名词为主。<span style="color:red">affect 偏重于施加影响的过程——动词（The weather affects your mood 天气影响你的心情）；effect 侧重于影响结束后留下的结果——名词（The effect of the weather on your mood 天气对你心情的影响）。</span><span style="color:red">一句话：affect 是「影响了」，effect 是「影响的结果」。</span>

**11. afford** 动词 <span style="color:blue">负担得起；提供</span>

- 构词：af-（<span style="color:blue">朝向</span>）+ forth（<span style="color:blue">向前</span>）

- 实际义：有足够的钱或时间来做或购买某物（负担得起）；正式用语——提供、给予（afford an opportunity）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| can/can't afford | 负担得起/不起 |
| afford to do sth | 有能力做某事 |
| afford an opportunity | 提供机会 |

- 字面义：<span style="color:red">站到旁边——走过去站在旁边随时搭把手</span>
- 实际义：通过提供帮助或支持让别人做某事更容易
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| assist sb with sth | 帮助某人做某事 |
| assist in doing | 协助做…… |
| assisted living | 辅助生活（养老） |

## 模块 5：终极避坑 + 词间连接

### 拼写变化规则

ad- 的变体来自<span style="color:red">拉丁内部的辅音同化</span>——同化的是 -d。

| 表面形式 | 触发条件 | 机制 | 例词 |
|---------|---------|------|------|
| **ad-** | 元音、d、h、j、m、v 前 | -d 保持 | adapt, adopt, admire, adjust |
| **a-** | sc、sp、st 前 | -d 完全脱落 | ascend, aspire |
| **ac-** | c、q 前 | -d → -c | accept, acquire |
| **af-** | f 前 | -d → -f | affect, afford |
| **ag-** | g 前 | -d → -g | aggression |
| **al-** | l 前 | -d → -l | allow |
| **an-** | n 前 | -d → -n | announce |
| **ap-** | p 前 | -d → -p | apply, appoint |
| **ar-** | r 前 | -d → -r | arrive |
| **as-** | s 前 | -d → -s | assist, associate |
| **at-** | t 前 | -d → -t | attend, attract |

同化逻辑：-d 是舌尖塞音。在后接辅音前，-d 调整自己的发音去匹配后面的辅音——变成同一个辅音（-d → -c/-f/-g/-l/-n/-p/-r/-s/-t），在 sc/sp/st 前直接脱落。和 con-（同化 -m）、syn-（同化 -n）属于同一类机制——<span style="color:red">都是舌头偷懒，只是偷懒的对象不同。</span>

### 反直觉的拼写——中世纪的"整形手术"

<span style="color:red">ad- 经历过一次被学者人为"纠正"的过程。</span>古法语把所有 ad- 变体统一简化成了 a-。15 世纪，英国抄写员觉得 a- 不够"拉丁"，按拉丁原文把 -d 拼了回去。结果：

- adjust 本该是 ajust（法语至今仍是 ajuster）
- advance 本该是 avance（法语 avancer）——而且 advance 根本就不是 ad- 词（来自拉丁 abante）
- admiral 来自阿拉伯语 amīr，和 ad- 无关，也被错加了一个 d

### 假词根甄别

| 单词 | 误解 | 真相 |
|------|------|------|
| **advance** | 以为是 ad- | 来自拉丁 abante（从前面），不是 ad- |
| **admiral** | 以为是 ad- | 来自阿拉伯语 amīr（指挥官），与 ad- 无关 |
| **a-（否定）** | 以为是 ad- 变体 | amoral 的 a- 是希腊否定前缀，ascend 的 a- 是 ad- 在 sc 前脱落 |
