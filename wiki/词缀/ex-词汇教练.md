## 模块 1：基础信息

**变体总览**——ex- 在英语里有 <span style="color:red">5 种表面形式</span>，来自 <span style="color:red">1 个 PIE 根 × 3 条语族路径</span>。变体由后接辅音决定——不是 cap- 那种 ablaut，而是<span style="color:red">语音同化</span>。

| 形式 | 出现条件/来源 | 代表词 |
|------|-------------|--------|
| **ex-** | 元音或清辅音前 | exit, export, exclude, extend |
| **e-** | 浊辅音（b, d, g, l, m, n, r, v）前，x 脱落 | emit, evolve, educate, emerge |
| **ef-** | f 前，x 被 f 同化 | effect, effort, efficient |
| **ec-** | 希腊借词路径 | eccentric, ecstasy |
| **es-** | 古法语路径 | escape, strange, essay |

一句话总结：共 5 种表面形式，核心只有 ex- 一个——其余全是它在不同语音环境下的变形。不是两个前缀，是同一个人换了五套衣服。

**核心本义**：<span style="color:blue">从内部向外移动——跨过一个边界，从里面到外面</span>

**同源词概览**：
- 拉丁线（借词）：英语中绝大多数 ex- 词，文艺复兴后批量直借
- 希腊线（借词）：exotic, ecstasy, eccentric——同一个 PIE 根走希腊路径
- 法语线（借词）：escape, strange, essay——经诺曼征服后进入，拼写中保留了古法语 es-
- 日耳曼线：<span style="color:red">无</span>——PIE \*eghs- 在日耳曼语族中没有留下常用同源词。这是 ex- 与 in- 的关键区别（in- 在日耳曼有 in + un- 两个后代）

**语域层**：
- 拉丁学术层：exclude, extract, elaborate, exaggerate
- 法语中性层：escape, strange, essay, effort
- 日常层：exit, extra, strange, effort
- 口语层：ex-wife, ex-president（19 世纪英语内部创新）

**词频层**：
- 第 1-2 层核心高频：exit, express, extra, effect, effort, extreme, strange, example, expect, explain, event, evidence
- 第 3-4 层中高频：export, extend, exclude, expand, exceed, excite, evolve, educate, emerge, enormous, eliminate, explore, expose, exhibit, exhaust, extract, exterior, exotic, erase, eject, elaborate, erect
- 第 5-6 层中低频：exclaim, exaggerate, evade, eccentric, ecstasy
- 第 7+ 层废弃/罕见：excruciate, expiate, exarch

## 模块 3：语义锚点 + 知识挂点

**知识挂点**：
- 你已经在用 exit（<span style="color:blue">出口</span>）——ex- + ire（<span style="color:blue">走</span>），「走出去」。这是 ex- 最直观的锚点。
- 学过法语：étranger（<span style="color:blue">外国的、陌生的</span>）和英语 strange 同源——都来自拉丁 extraneus（<span style="color:blue">来自外面的</span>），古法语 es- → 英语 s 开头。
- 学过法语：essayer（<span style="color:blue">尝试</span>）和英语 essay 同源——es- + agere（<span style="color:blue">做</span>），「尝试做出来」。
- 中文母语者：中文「出」的空间逻辑和 ex- 几乎完全对应——「走出去」= exit、「运出去」= export、「关出去」= exclude。ex- = 英语的「出」。

**底层核心逻辑**（<span style="color:red">红色</span>）：ex- 的所有现代义，回到同一个空间经验——<span style="color:blue">"从一个有边界的空间内部，穿过边界，到达外部"</span>。

<span style="color:red">物理出界</span>（exit 走出门、export 运出国境、extract 拔出来）→ <span style="color:red">排斥在外</span>（exclude 关在门外、eliminate 赶出门外）→ <span style="color:red">展开呈现</span>（explain 铺平展开、express 压出来、expose 放出去）→ <span style="color:red">超越极限</span>（exceed 走出上限、extreme 走到尽头、exaggerate 堆过头）→ <span style="color:red">彻底完成</span>（effect 做出来、exhaust 彻底抽干、elaborate 彻底做出来）→ <span style="color:red">身份脱离</span>（ex-wife 从婚姻中走出来、escape 从斗篷中脱出）。

六个语义层，同一个空间动作。

**易误解盲区**：
- ex- 词的第二个成分<span style="color:red">不是 ex- 的变体</span>，而是<span style="color:red">独立的动词词根</span>。extract = ex-（<span style="color:blue">向外</span>）+ trahere（<span style="color:blue">拉</span>），trahere 是一个完整的拉丁动词。这与 cap- 完全不同——cap- 的 -cip-、-cept-、-ceiv- 全都是 cap- 自身的变形。
- expect、example、examine 三个词表面上不像 ex- 词，但确实都是：expect = ex- + spectare（<span style="color:blue">向外看→期待</span>）；example = ex- + emere（<span style="color:blue">拿出来→样本</span>）；examine = ex- + agere（<span style="color:blue">往外称→仔细检查</span>）。<span style="color:red">不要因为拼写面熟就忽略了 ex- 的存在。</span>
- ⚠️ **em- 双重身份**：emerge, emit 的 e- 是 ex- 的变体（<span style="color:blue">向外</span>）；但 embody, empower 的 em- 是 en-（<span style="color:blue">向内</span>）在 b/p/m 前的变体——两个无关前缀，同一个表面形式。鉴别法：<span style="color:red">em- + 名词/形容词 → en- 向内（embody, empower, embitter）；em- + 动词词根 → ex- 向外（emerge, emit）</span>。

## 模块 4：高频词汇拆解

从速览表 45 词中精选 22 个高频词。按变体形式分为三组。优先选有近义辨析价值的词。

### A 组：ex- 标准形（12 词）

拉丁 ex- 在元音和清辅音前保持完整形式。

**1. exit** 名词/动词 <span style="color:blue">出口；退出；离去</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ ire（<span style="color:blue">走</span>）
- 字面义：<span style="color:red">走出去</span>
- 实际义：名词——建筑物或高速公路的出口；动词——离开某个地方、退出程序或组织
- 词频层：第 1 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| emergency exit | 紧急出口 |
| exit the building | 离开建筑物 |
| exit strategy | 退出策略 |

**2. exclude** 动词 <span style="color:blue">排除；排斥；不让进入</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ claudere（<span style="color:blue">关、闭</span>）
- 字面义：<span style="color:red">关在外面</span>
- 实际义：故意把某人或某物排除在某个范围之外；阻止某人参与或进入
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| exclude sb from sth | 把某人排除在某事之外 |
| feel excluded | 感到被排斥 |
| excluding tax | 不含税 |

**3. extend** 动词 <span style="color:blue">延伸；扩展；提供</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ tendere（<span style="color:blue">伸展</span>）
- 字面义：<span style="color:red">向外伸展出去</span>
- 实际义：在空间或时间上拉长（延伸）；把范围扩大到更多人/事物（扩展）；主动把某物递给某人（提供、给予）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| extend a deadline | 延长截止日期 |
| extend an invitation | 发出邀请 |
| extend beyond | 超出……范围 |

- vs expand：<span style="color:blue">都作「变大变多」解</span>。extend = ex-（<span style="color:blue">向外</span>）+ tendere（<span style="color:blue">伸展</span>）——沿着一条线「伸」出去；expand = ex-（<span style="color:blue">向外</span>）+ pandere（<span style="color:blue">展开</span>）——把折叠的东西「展」开。<span style="color:red">extend 偏重于线性延伸——一个方向拉长（时间、空间、范围）；expand 侧重于多维膨胀——所有方向同时扩大（体积、规模、业务）。</span>公路 extend（拉长），气球 expand（鼓起来）。

**4. expand** 动词 <span style="color:blue">膨胀；扩展；详述</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ pandere（<span style="color:blue">展开、铺开</span>）
- 字面义：<span style="color:red">把折叠的东西展开</span>
- 实际义：体积或尺寸变大（膨胀）；业务或活动范围扩大（扩展）；把简短的观点说详细（详述、展开论述）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| expand business | 扩展业务 |
| expand on a point | 详细阐述一个观点 |
| metals expand when heated | 金属受热膨胀 |

- vs extend：<span style="color:red">见 extend。</span>

**5. express** 动词/形容词 <span style="color:blue">表达；快递的；明确的</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ pressare（<span style="color:blue">压、挤</span>）
- 字面义：<span style="color:red">从内部压出来</span>
- 实际义：用语言或行为把内心的想法/感情「压出来」传达给他人（表达）；以最快速度运送——中途不「压」停靠站（快递的、特快的）；清楚而直接地说出（明确的）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| express oneself | 表达自己 |
| express train | 特快列车 |
| express purpose | 明确目的 |

**6. exceed** 动词 <span style="color:blue">超过；超出</span>

- 构词：ex-（<span style="color:blue">向外、超出</span>）+ cedere（<span style="color:blue">走</span>）
- 字面义：<span style="color:red">走出限度之外</span>
- 实际义：数量或数值超过某个标准；超出预期或允许的范围
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| exceed expectations | 超出预期 |
| exceed the speed limit | 超速 |
| exceed the budget | 超出预算 |

- vs surpass：<span style="color:blue">都作「超过」解</span>。exceed = ex-（<span style="color:blue">向外</span>）+ cedere（<span style="color:blue">走</span>）——从限度的线「走」出去——中性，可好可坏（exceed the speed limit 是坏事）；surpass = sur-（<span style="color:blue">在上面</span>）+ passer（<span style="color:blue">走过</span>）——从「上面」「走过」去——自带「优于、胜过」的正面评价。<span style="color:red">exceed 偏重于超过客观数字或限度——标准是量化的；surpass 侧重于在品质或能力上超越——标准是评价性的。</span>超速用 exceed不用 surpass，打破了记录两者皆可。

**7. extract** 动词/名词 <span style="color:blue">提取；摘录；拔除</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ trahere（<span style="color:blue">拉、拖</span>）
- 字面义：<span style="color:red">从内部用力拉出来</span>
- 实际义：从某物内部把东西用力取出（提取、萃取）；从文本中选出一段（摘录）；拔出牙齿等（拔除）
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| extract a tooth | 拔牙 |
| extract data | 提取数据 |
| an extract from a novel | 小说节选 |

**8. extreme** 形容词/名词 <span style="color:blue">极端的；极度的；极端</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ PIE 最高级后缀 \*-t(e)r-emo-（<span style="color:blue">最…的</span>）
- 字面义：<span style="color:red">最外面的、走到最尽头的</span>
- 实际义：达到最大程度的（极度的）；观点或行为远离常规的（极端的）；完全相反的另一端（名——极端）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| extreme weather | 极端天气 |
| go to extremes | 走极端 |
| extreme caution | 极度谨慎 |

**9. exterior** 名词/形容词 <span style="color:blue">外部；外观；外面的</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ 拉丁比较级后缀 -terior（<span style="color:blue">更…的</span>）
- 字面义：<span style="color:red">更外面的——与 interior（更里面的）互为反义</span>
- 实际义：建筑物或物体的外部表面（外部、外观）；人表现给外界看的样子（外在的）
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| exterior wall | 外墙 |
| exterior design | 外观设计 |
| a calm exterior | 冷静的外表 |

**10. explain** 动词 <span style="color:blue">解释；说明</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ planare（<span style="color:blue">使平坦、铺平</span>）
- 字面义：<span style="color:red">把褶皱铺平展开</span>
- 实际义：把复杂的事情「铺平」让人看清楚——用言语把道理讲明白
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| explain why / how | 解释为什么 / 如何 |
| explain oneself | 为自己的行为辩解 |
| explain away | 搪塞过去 |

**11. exaggerate** 动词 <span style="color:blue">夸张；夸大</span>

- 构词：ex-（<span style="color:blue">超出</span>）+ aggerare（<span style="color:blue">堆积、堆高</span>）
- 字面义：<span style="color:red">堆过头——把事实堆得比实际大</span>
- 实际义：把事物描述得比实际情况更大、更严重、更好或更极端
- 词频层：第 5 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| exaggerate the difficulty | 夸大困难 |
| it's hard to exaggerate | 怎么说都不为过 |
| grossly exaggerate | 极度夸大 |

- vs overstate：<span style="color:blue">都作「夸大」解</span>。exaggerate = ex-（<span style="color:blue">超出</span>）+ aggerare（<span style="color:blue">堆积</span>）——「堆」过头——把原本一小堆土堆成一座山——暗示有添油加醋的动作；overstate = over-（<span style="color:blue">过度</span>）+ state（<span style="color:blue">陈述</span>）——「陈述过度」——把程度说高了，但不暗示编造。<span style="color:red">exaggerate 偏重于在事实基础上添油加醋——有虚构和膨胀的成分；overstate 侧重于把真实情况的程度说高了——不一定是编造，可能是措辞问题。</span>他说「我快死了」是 exaggerate（编造极端后果），他说「问题很严重」（而实际上中等）是 overstate。

**12. exotic** 形容词 <span style="color:blue">异国的；奇异的；异国情调的</span>

- 构词：ex-（<span style="color:blue">向外</span>）+ 希腊 exō（<span style="color:blue">外面</span>）+ -tic（<span style="color:blue">形容词后缀</span>）
- 字面义：<span style="color:red">来自外面的——不是本地的、从遥远的外部来的</span>
- 实际义：来自遥远异国的事物，因其不常见而显得有趣、迷人或有异国情调
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| exotic food | 异国美食 |
| exotic location | 异国风情的目的地 |
| exotic species | 外来物种 |

- vs foreign：<span style="color:blue">都作「外国的」解</span>。exotic = ex-（<span style="color:blue">向外</span>）+ 希腊 exō（<span style="color:blue">外面</span>）——「来自外面的」——强调「不常见」带来的新奇和吸引力；foreign = 拉丁 foris（<span style="color:blue">门外</span>）——「门外的」→ 不属于本国的。<span style="color:red">exotic 偏重于因遥远和不常见而显得迷人、有异域魅力——带正面审美色彩；foreign 侧重于客观的国籍或来源归属——中性的行政标签。</span>一盘泰国菜是 exotic food（异国风味），一本外国护照是 foreign passport（外国护照——不加审美判断）。

### B 组：e- 浊辅音前脱落（9 词）

拉丁 ex- 在 b, d, g, l, m, n, r, v 前，x 脱落变成 e-。这是「同一个人，嗓子哑了」——不是另一个前缀。

**13. emit** 动词 <span style="color:blue">发出；排放；发射</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 m 前的变体</span>）+ mittere（<span style="color:blue">送、放</span>）
- 字面义：<span style="color:red">送出去</span>
- 实际义：把光、热、声音、气体等从源头持续向外发送
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| emit light / heat | 发出光 / 热 |
| emit carbon dioxide | 排放二氧化碳 |
| emit a sound | 发出声音 |

**14. evolve** 动词 <span style="color:blue">进化；发展；逐步形成</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 v 前的变体</span>）+ volvere（<span style="color:blue">滚、卷</span>）
- 字面义：<span style="color:red">把卷着的东西向外滚开、展开</span>
- 实际义：生物种群在漫长时间中逐渐变化（进化）；某种制度或想法缓慢而自然地发展（逐步形成）
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| evolve over time | 随时间逐渐演变 |
| evolve into | 演变成 |
| evolve a strategy | 逐步形成策略 |

- vs develop：<span style="color:blue">都作「发展」解</span>。evolve = e-（<span style="color:blue">向外</span>）+ volvere（<span style="color:blue">卷</span>）——把内在已经「卷」着的东西「展开」——变化来自内部、自然展开；develop = des-（<span style="color:blue">解开</span>）+ veloper（<span style="color:blue">包裹</span>）——「把包裹解开」→ 让潜在的东西显现出来。<span style="color:red">evolve 偏重于内在驱动、缓慢自然的渐进变化——像一本书卷着自己展开；develop 侧重于有外力推动或人为规划的发展——从无到有、从小到大。</span>物种 evolve（自然演化），房地产 develop（人为开发）。

**15. emerge** 动词 <span style="color:blue">浮现；出现；显露</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 m 前的变体</span>）+ mergere（<span style="color:blue">沉入、浸入</span>）
- 字面义：<span style="color:red">从沉没中出来</span>
- 实际义：从水中或隐藏状态中浮出来；新事实或新问题开始被人注意到
- 词频层：第 3 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| emerge from | 从……中浮现出来 |
| emerging markets | 新兴市场 |
| it emerged that | 事实表明…… |

- vs appear：<span style="color:blue">都作「出现」解</span>。emerge = e-（<span style="color:blue">向外</span>）+ mergere（<span style="color:blue">沉入</span>）——之前是「沉」在水下的，现在「浮出来」——强调有一个从隐藏到可见的过程；appear = ad-（<span style="color:blue">朝向</span>）+ parere（<span style="color:blue">显现</span>）——「朝这边显现」→ 进入视野。<span style="color:red">emerge 偏重于从不可见到可见的渐进过程——出来之前就在那里，只是看不见；appear 侧重于突然进入视线——不暗示之前被隐藏。</span>太阳从云后 emerge（从遮住到露出），鬼魂出现在门口用 appear。

**16. educate** 动词 <span style="color:blue">教育；培养</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 d 前的变体</span>）+ ducere（<span style="color:blue">引导</span>）
- 字面义：<span style="color:red">把内在的能力引导出来</span>
- 实际义：通过系统的教学和训练培养人的知识和能力
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| educate children | 教育孩子 |
| educate the public | 教育公众 |
| educate oneself | 自学 |

**17. event** 名词 <span style="color:blue">事件；活动；比赛项目</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 v 前的变体</span>）+ venire（<span style="color:blue">来</span>）
- 字面义：<span style="color:red">出来的事情——从时间流中冒出来的事</span>
- 实际义：发生的某件重要事情；有组织的社会活动或集会；体育比赛中的一个项目
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| a historic event | 历史性事件 |
| sporting event | 体育赛事 |
| in the event of fire | 如果发生火灾 |

- vs incident：<span style="color:blue">都作「事件」解</span>。event = e-（<span style="color:blue">向外</span>）+ venire（<span style="color:blue">来</span>）——从时间流中「冒出来」的事——中性，可以是好事也可以是坏事；incident = in-（<span style="color:blue">在…上</span>）+ cadere（<span style="color:blue">落下</span>）——「落在头上的事」——偏向意外、不愉快。<span style="color:red">event 偏重于是计划中或值得注意的节点——有意义、有安排；incident 侧重于意外发生的插曲——通常是令人不快的偶发事件。</span>奥运开幕式是 event，走廊里撞到同事是 incident。

**18. evidence** 名词/动词 <span style="color:blue">证据；迹象；证明</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 v 前的变体</span>）+ videre（<span style="color:blue">看</span>）
- 字面义：<span style="color:red">从里面往外看——从模糊中看出来的东西</span>
- 实际义：能帮助你判断事情真相的信息（证据）；某事物存在或发生的可见迹象
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| there is evidence that | 有证据表明 |
| lack of evidence | 缺乏证据 |
| give evidence in court | 在法庭上作证 |

- vs proof：<span style="color:blue">都作「证据」解</span>。evidence = e-（<span style="color:blue">向外</span>）+ videre（<span style="color:blue">看</span>）——「看出来」的东西——指向真相的线索和信息，是探索和推断的工具；proof = 拉丁 probare（<span style="color:blue">测试、证明</span>）——「验证」的结果——经过检验后确认无误的结论。<span style="color:red">evidence 偏重于指向结论但本身不一定结论性的材料——可以积累，可以有强有弱（circumstantial evidence）；proof 侧重于充分到足以确立事实的证明——逻辑上或法律上已经定论。</span>侦探在犯罪现场收集 evidence，DNA 匹配才是 proof。

**19. erase** 动词 <span style="color:blue">抹去；删除；消除</span>

- 构词：e-（<span style="color:blue">彻底，ex- 在 r 前的变体</span>）+ radere（<span style="color:blue">刮、擦</span>）
- 字面义：<span style="color:red">彻底刮掉——连痕迹都不剩</span>
- 实际义：从纸张、磁带、记忆等载体上完全去除信息或痕迹
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| erase a mistake | 擦掉错误 |
| erase data | 删除数据 |
| erase from memory | 从记忆中抹去 |

- vs delete：<span style="color:blue">都作「删除」解</span>。erase = e-（<span style="color:blue">彻底</span>）+ radere（<span style="color:blue">刮</span>）——「刮」掉——像用橡皮擦在纸上擦，物理上把痕迹磨掉；delete = 拉丁 delere（<span style="color:blue">摧毁、消灭</span>）——「删除」——勾掉一行字，数据还在但不可访问。<span style="color:red">erase 偏重于物理性的去除——痕迹本身被破坏、不可恢复（橡皮擦擦铅笔字）；delete 侧重于逻辑性的移除——数据标记为删除但可能可恢复（电脑上的 delete 键）。</span>erase a whiteboard（擦白板——物理动作），delete a file（删除文件——逻辑操作）。

**20. eject** 动词 <span style="color:blue">弹出；驱逐；喷射</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 j 前的变体</span>）+ jacere（<span style="color:blue">扔、投掷</span>）
- 字面义：<span style="color:red">扔出去——把人或物从内部用力抛到外部</span>
- 实际义：用机械力把某物从设备中弹出；强制某人离开某个场所；飞机紧急情况下把飞行员弹射出去
- 词频层：第 4 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| eject a CD | 弹出光盘 |
| be ejected from the bar | 被赶出酒吧 |
| ejector seat | 弹射座椅 |

- vs expel：<span style="color:blue">都作「驱逐」解</span>。eject = e-（<span style="color:blue">向外</span>）+ jacere（<span style="color:blue">扔</span>）——「扔」出去——用力抛，动作猛烈、一次性；expel = ex-（<span style="color:blue">向外</span>）+ pellere（<span style="color:blue">推、驱赶</span>）——「推」出去——用推力驱赶，带有制度或权力的意味。<span style="color:red">eject 偏重于物理上的猛然抛出——像弹射座椅那样把人射出去；expel 侧重于制度性的驱逐——学校开除学生、国家驱逐外交官。</span>CD 从播放器 eject（机械弹出），学生被学校 expel（开除学籍）。

**21. evade** 动词 <span style="color:blue">逃避；躲避；规避</span>

- 构词：e-（<span style="color:blue">向外，ex- 在 v 前的变体</span>）+ vadere（<span style="color:blue">走、去</span>）
- 字面义：<span style="color:red">走出去→从该在的地方溜走——该面对却绕开了</span>
- 实际义：用巧妙或狡猾的方式逃避该面对的人、责任、问题或法律义务
- 词频层：第 5 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| evade taxes | 逃税 |
| evade the question | 回避问题 |
| evade capture | 逃脱抓捕 |

### C 组：ef- f 前同化（1 词）

**22. effect** 名词/动词 <span style="color:blue">效果；影响；作用；实现</span>

- 构词：ef-（<span style="color:blue">彻底，ex- 在 f 前的变体</span>）+ facere（<span style="color:blue">做</span>）
- 字面义：<span style="color:red">彻底做出来→造成的结果</span>
- 实际义：某个原因导致的后果或变化（效果、影响）；用于某种特定目的产生的结果（作用）；作为动词——正式用语，使某事发生（实现、达成）
- 词频层：第 2 层
- 搭配：

| 搭配模式 | 说明 |
|---------|------|
| cause and effect | 因果 |
| have an effect on | 对……有影响 |
| side effects | 副作用 |
| effect a change | 实现变革 |

- vs affect：<span style="color:blue">都作「影响」解</span>——这是英语中最经典的近义词混淆。effect = ef-（<span style="color:blue">彻底</span>）+ facere（<span style="color:blue">做</span>）——「彻底做出来」→ 名词为主——做出来的「结果」；动词——「把某事做成功、使发生」。affect = ad-（<span style="color:blue">朝向</span>）+ facere（<span style="color:blue">做</span>）——「朝某个方向做」→ 动词为主——「施加影响于……」。<span style="color:red">effect 偏重于作为名词的「结果」——做完之后留下来的东西；动词义极其正式（effect a change = 实现变革）；affect 侧重于作为动词的「施加影响的过程」——对某人或某事产生作用。</span><span style="color:red">一句话：affect 是「影响了」，effect 是「影响的结果」和「使……发生」。</span>

## 模块 5：终极避坑 + 词间连接

### 拼写变化规则

ex- 不涉及 PIE 元音变换（不是动词词根，没有 ablaut）。它的变体来自<span style="color:red">拉丁内部的辅音同化</span>——这是 ex- 与 cap- 的根本技术区别。

| 表面形式 | 触发条件 | 机制 | 例词 |
|---------|---------|------|------|
| **ex-** | 元音或清辅音前 | x 保持不变 | exit, export, exclaim, extend |
| **e-** | 浊辅音（b, d, g, l, m, n, r, v）前 | x（= /ks/）在浊音前难以保持，整体脱落 | emit, educate, evolve, emerge, erase, eject, evade, elaborate, enormous, eliminate, event, evidence, erect |
| **ef-** | f 前 | x 的 /s/ 被唇擦音 f 同化——嘴唇懒得从 /s/ 切换到 /f/ | effect, effort, efficient |
| **ec-** | 希腊借词中 | 希腊语 ek-（ex- 的希腊形式）在英语中的残留 | eccentric, ecstasy |
| **es-** | 古法语路径 | 拉丁 ex- 在古法语中弱化为 es-，经诺曼征服进入英语 | escape, strange, essay |

同化逻辑：x = /ks/。发音时舌头的动作是：/k/（舌根抵软腭）→ /s/（舌尖抵齿槽）。在浊辅音前，整个 /ks/ 音丛难以保持，x 脱落。在 f（唇擦音）前，/s/ 被 f 的回声同化——嘴唇来不及从 /s/ 的唇形切换到 /f/ 的唇形。这不是深奥的 PIE 语法，是说话时舌头和嘴唇偷懒的结果。六千年后，这些偷懒的痕迹变成了你必须记的拼写规则。

### ex- 与其他词根的根本区别

<span style="color:red">ex- 是「前缀型」词根——这是它与 cap- 等「动词型」词根的根本区别。</span>

| | 前缀型（ex-） | 动词型（cap-） |
|------|------|------|
| 原始身份 | PIE 小品词/介词 | PIE 动词词根 |
| 变体来源 | 语音同化（后接辅音决定） | PIE ablaut（元音等级） |
| 后接成分 | ex- + 独立动词词根 | 前缀 + cap- 自身的变体 |
| 例 | extract = ex- + trahere（独立动词） | accept = ad- + cap- 变体 -cept- |
| 日耳曼同源 | <span style="color:red">无</span> | 有（have, heave, heavy） |
| 英语新造词 | 有——19 世纪 ex- 独立成词（ex-wife） | 无——已死亡 |

### 假词根甄别

ex- 几乎没有假词根——所有 ex-/e-/ef- 开头的拉丁词基本都来自 PIE \*eghs。主要的坑不是假词根，而是前缀混淆：

| 单词 | 误解 | 真相 |
|------|------|------|
| **expect** | 不像 ex- 词，以为 ex- 没有 | ex- + spectare——「向外看」 |
| **example** | 不像 ex- 词，以为 exam- 是一个单位 | ex- + emere——「拿出来」 |
| **examine** | 不像 ex- 词 | ex- + agere——「往外称」 |
| **emerge / emit** | 以 em- 开头，以为是 en- 的变体 | 是 ex- 在 m 前的变体——「向外」 |
| **embody / empower** | 以 em- 开头，跟 emerge 同一个表面形式 | 是 en-（向内）的变体——跟 ex- 无关 |

<span style="color:red">快速鉴别法：em- + 名词/形容词 → en- 向内（embody, empower）；em- + 动词词根 → ex- 向外（emerge, emit）。</span>
