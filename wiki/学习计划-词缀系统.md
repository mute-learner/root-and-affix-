## 数据依据

分组按**实际内容量**而非语义分组，确保每天工作量平衡。数据来源：Haldeman（词典衍生词统计）+ White/Sowell/Yanagihara（印刷英语频率）。

| 体量级 | 前缀 | 衍生词约数 | 频率占比 | 说明 |
|--------|------|-----------|---------|------|
| 🔴 巨 | un- | ~5,600 | 26% | 频率之王，但形式单一 |
| 🔴 巨 | in- 双义 | ~2,900 | 11%+4% | 两个 PIE 源头 + 三变体 |
| 🔴 巨 | con- | ~2,400 | <1% | 四变体 co-/com-/col-/cor- |
| 🔴 巨 | re- | ~2,200 | 14% | 频率 #2，无变体 |
| 🔴 巨 | dis- | ~1,800 | 7% | 双义 + 变体 di-/dif- |
| 🟡 大 | ex- | ~1,750 | <1% | 4 变体，已有完整笔记 |
| 🟡 大 | ad- | ~1,600 | <1% | 12 个同化变体 |
| 🟡 大 | de- | ~1,600 | 2% | 三义，无变体 |
| 🟢 中 | sub- | ~700 | 3% | 7 变体 |
| 🟢 轻 | 其余 21 个 | <500 各 | 1-3% 各 | 形式简单，变体少 |

**后缀体量**：-tion/-ment/-al/-ure 需要四向辨析（最重），人物后缀五人组次之，形容词后缀多为一行定义 + 两个例词（最轻）。

## 分组原则

1. **体量平衡**——每天前缀总衍生词量 + 后缀辨析深度大致相当
2. **对比价值**——同一组内的前缀/后缀有自然的反义或近义关系
3. **前缀后缀各自独立分组**——不强行混排。同一天前缀组和后缀组的主题可能无关，各有各的故事线

## 8 天学习计划（每天 2 组）

### 第 1 天：in- 双面人 + ex- 复习

**前缀（2 个，🔴 体量）：** in-双义 + ex-复习

> <span style="color:red">in- 的双重身份</span>——同一个形式 in-/im-/il-/ir-，有时候是「向内」(import)，有时候是「不」(impossible)。两个完全不同的 PIE 词根，必须一次性讲透。ex- 已有 [[ex-词汇教练]] + [[ex-词源史官]]，今天只做快速回顾。

**后缀：无**（第一天重心在 in-）

| 内容 | 体感 |
|------|------|
| in-双义的两种 PIE 源头 + 三种语音变体 + 向内/否定衍生词全景 | **重** — 深度理解 |
| ex- 四变体 + 核心衍生词回顾 | 轻 — 已有笔记 |

---

### 第 2 天：con- + syn-——「一起」的拉丁线与希腊线

**前缀（2 个，🔴 体量）：** con- + syn-

> con- 是拉丁线「一起」（<span style="color:red">四个变体</span> co-/com-/col-/cor-），syn- 是希腊线「一起」（三个变体 sym-/syl-/sys-）。同义双胞胎，但进入英语的路径完全不同。

**后缀：无**

| 内容 | 体感 |
|------|------|
| con- 四变体的同化规则 + 大量衍生词（connect, combine, collect, correct...） | **重** |
| syn- 三变体 + 派生词（synonym, sympathy, syllable, system） | 轻 |

---

### 第 3 天：ad- + ab- + se-——方向三态

**前缀（3 个，🟡🔴 体量）：** ad- + ab- + se-

> ad-（朝向，<span style="color:red">12 个同化变体</span>——整个前缀系统里最复杂的单项），ab-（离开），se-（分离）。往哪里去 + 从哪里离开 + 横向分离——三个方向操作。

**后缀：无**

| 内容 | 体感 |
|------|------|
| ad- 12 变体的同化逻辑（ac-/af-/ag-/al-/an-/ap-/ar-/as-/at-） | **重** — 单项最复杂 |
| ab-（abs-/a-）+ se-（少数衍生词） | 轻 |

---

### 第 4 天：dis- + re- + de-——动作的三个方向

**前缀（3 个，🔴🔴🟡 体量）：** dis- + re- + de-

> <span style="color:red">最重的一天。</span>dis-（分开/否定，#4 频率），re-（回/再/对抗，#2 频率），de-（向下/脱离/彻底）。三个都高频、衍生词极多，但结构不复杂（变体少）——重在工作量，不在理解难度。

**后缀：无**

| 内容 | 体感 |
|------|------|
| dis- 双义 + 变体 di-/dif- + 高频衍生词 | **重** |
| re- 三义 + 超高频衍生词 | **重** |
| de- 三义 + 中高频衍生词 | 中 |

---

### 第 5 天：时间 + 穿过（轻量前缀收割）+ 动作名词四向辨析

**前缀（6 个，🟢 体量）：** pre- + post- + pro- + trans- + per- + dia-

> 六个轻量前缀。pre-/post- 是时间线反义对，pro- 补「向前」维度。trans-/per-/dia- 是「穿过」三向对比——横穿/穿透/通过。每个结构简单、变体极少，六合一仍轻松。

**后缀（4 个，🔴 体量）：** -tion, -ment, -al, -ure

> 四个都表「动作/过程/结果」，<span style="color:red">整个后缀系统里最容易混的一组</span>。decide → decision，move → movement，arrive → arrival，fail → failure——为什么同一个动词接不同的名词后缀？

| 内容 | 体感 |
|------|------|
| 6 个轻量前缀（表格为主，每个 10-15 分钟） | 中 |
| -tion vs -ment vs -al vs -ure 四向辨析 | **重** — 后缀系统核心 |

---

### 第 6 天：否定家族 + 人物后缀五人组

**前缀（4 个，🔴🟢 体量）：** un- + non- + mis- + a-

> un- 是频率之王（26%），但形式单一、含义直白。non-（中性否定）、mis-（错误）、a-（希腊式「缺乏」）三个轻量搭配，一天刚好。

**后缀（5 个，🟡 体量）：** -er/-or, -ist, -ee, -ant/-ent, -ian

> 五种「做…的人」。teach-er（日耳曼日常），act-or（拉丁专业），art-ist（主义者），employ-ee（被…的人），assist-ant（正在做…的人），music-ian（专业人员）。区别不在含义，在语域和隐含态度。

| 内容 | 体感 |
|------|------|
| un- 双义（否定 + 反转动作）+ 三个轻量否定 | 中 |
| 人物后缀五人组对比 | 中 — 表格 + 语域辨析 |

---

### 第 7 天：空间位置 + 反对 + 抽象名词六件套

**前缀（7 个，🟢 体量）：** sub- + super- + inter- + circum- + anti- + contra- + ob-

> 七个轻量前缀。上下（super-/sub-）、之间（inter-）、周围（circum-）、反对（anti-/contra-/ob-）——覆盖最后一批空间和关系前缀。数量多但每个内容少。

**后缀（6 个，🟡🟢 体量）：** -ness, -ity, -th, -ism, -ship, -hood

> 抽象名词六件套。-ness（日耳曼）vs -ity（拉丁）是核心对比——happiness vs reality，同一功能，两个语族。-th 是日耳曼最老的抽象后缀（warmth, health）。-ism/-ship/-hood 覆盖主义/关系/身份。

| 内容 | 体感 |
|------|------|
| 7 个轻量前缀（表格为主） | 中 |
| -ness vs -ity 日耳曼/拉丁对比 + 其余四个表格 | 中 |

---

### 第 8 天：独立前缀 + 形容词/动词/补充后缀收割

**前缀（3 个，🟢 体量）：** en- + auto- + meta-

> 三个没有天然对比关系的独立前缀。en-（使进入某状态），auto-（自己），meta-（超越/关于自身）。各自简述，不强行对比。

**后缀（18 个，🟢 体量）：**

动作名词补完：**-age, -ance/-ence, -ing**（轻量，补充第 5 天的动作名词体系）

形容词后缀拉丁线：**-able/-ible, -ive, -al/-ial, -ous/-ious, -ic/-ical**（表格速览，都表「…的」，区别在 nuance）

形容词后缀日耳曼线：**-ful, -less, -ish, -y, -ly(形)**（表格速览，直觉可懂）

动词后缀三向辨析：**-ate, -ify, -ize**（唯一需要展开的——都表「使…化」，为什么 active → activate 而不是 activize？）

副词后缀：**-ly(副)**（一行结束）

> <span style="color:red">最后一天做收割，不是深挖。</span>形容词后缀都只表「…的」，每个两行（一个例词 + 一个规则）就够。动词后缀三向辨析是唯一需要展开的内容。

| 内容 | 体感 |
|------|------|
| 3 个独立前缀 | 轻 |
| 动作名词补完（3 个） | 轻 |
| 形容词后缀拉丁线（5 个，表格） | 中 — 数量多但每项轻 |
| 形容词后缀日耳曼线（5 个，表格） | 轻 — 直觉可懂 |
| 动词后缀 -ate vs -ify vs -ize | 中 — 三向辨析展开 |
| 副词 -ly | 极轻 |

## 产出格式

每组产出一份词汇教练笔记。沿用现有体系：YAML frontmatter → 模块 1-4 → 三色规则。

注意：同一组内的多个前缀/后缀共享一份笔记，模块 1 的变体总览表覆盖多个词缀。模块 4（终极避坑 + 词间连接）是每一份笔记的灵魂——自然地利用组内对比关系做辨析。

节奏：每天 2 组（可按前缀组先行、后缀组后行，顺序自由），8 天完成全部 30 前缀 + 32 后缀。
