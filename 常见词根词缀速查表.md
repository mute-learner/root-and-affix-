# 常见英语词根词缀速查表

> 整理日期：2026-07-24

---

## 一、常用前缀 (Prefixes)

| 前缀 | 含义 | 示例 |
|------|------|------|
| **un-** | 不、相反 | unhappy (不开心), unable (不能) |
| **re-** | 再、重新 | rewrite (重写), review (复习) |
| **pre-** | 在前、预先 | preview (预览), prefix (前缀) |
| **dis-** | 否定、相反 | disagree (不同意), disappear (消失) |
| **mis-** | 错误 | mistake (错误), misunderstand (误解) |
| **over-** | 过度 | overuse (过度使用), overload (超载) |
| **sub-** | 下、次 | subway (地铁), substandard (次标准) |
| **inter-** | 之间、相互 | international (国际的), interact (互动) |

## 二、常用词根 (Roots)

| 词根 | 含义 | 示例 |
|------|------|------|
| **port** | 携带 | transport (运输), import (进口), export (出口) |
| **spect** | 看 | inspect (检查), respect (尊敬), spectator (观众) |
| **dict** | 说 | predict (预测), contradict (反驳), dictionary (词典) |
| **struct** | 建造 | construct (建造), structure (结构), destroy (摧毁) |
| **ject** | 投掷 | reject (拒绝), inject (注射), project (投射) |
| **form** | 形状 | transform (转变), uniform (制服), reform (改革) |
| **rupt** | 打破 | interrupt (打断), bankrupt (破产), disrupt (扰乱) |
| **vis** | 看 | visible (可见的), vision (视力), supervise (监督) |
| **graph** | 写、画 | photograph (照片), biography (传记), autograph (签名) |
| **log** | 说、学 | dialogue (对话), biology (生物学), logic (逻辑) |

## 三、常用后缀 (Suffixes)

### 名词后缀
| 后缀 | 含义 | 示例 |
|------|------|------|
| **-tion/-sion** | 动作/状态 | education, decision, information |
| **-ment** | 结果/行为 | movement, agreement, development |
| **-ness** | 性质/状态 | happiness, kindness, darkness |
| **-er/-or** | 做某事的人 | teacher, actor, worker |
| **-ist** | 专家/信仰者 | artist, scientist, dentist |

### 形容词后缀
| 后缀 | 含义 | 示例 |
|------|------|------|
| **-ful** | 充满 | beautiful, helpful, wonderful |
| **-less** | 没有 | homeless, careless, endless |
| **-able/-ible** | 能够 | readable, comfortable, visible |
| **-ous** | 具有…性质的 | dangerous, famous, nervous |
| **-ive** | 倾向/性质 | active, creative, expensive |

### 动词后缀
| 后缀 | 含义 | 示例 |
|------|------|------|
| **-ize/-ise** | 使…化 | realize, organize, apologize |
| **-en** | 使… | strengthen, widen, deepen |

## 四、记忆小技巧

1. **词根+词缀组合法**：掌握常见词根和词缀，可以推测陌生单词的含义。
   - 例如：`un` (不) + `believ` (相信) + `able` (能够) = **unbelievable** (难以置信的)

2. **联想记忆**：将词根与中文谐音联想，帮助记忆。
   - 例如：`port` (港口) → 从港口**携带**货物 → 携带

3. **分类整理**：按主题或词根分类，建立个人词根词缀库。
